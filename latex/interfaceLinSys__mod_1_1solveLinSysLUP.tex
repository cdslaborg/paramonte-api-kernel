Solve the set of {\ttfamily nd} linear equations of the form,.  




\doxysubsection{Detailed Description}
Solve the set of {\ttfamily nd} linear equations of the form,. 

\begin{equation} A \cdot X = B \end{equation} where $A$ is the coefficient matrix represented by its LU-\/\+Pivoted decomposition matrix {\ttfamily Coef\+Mat\+LUP}, $X$ is the output solution, and $B$ is the right-\/hand-\/side vector of constants.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Solution} & \+: The input contiguous vector of size {\ttfamily nd} of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}, representing the right-\/hand-\/side $B$ in the system of linear equations to be solved. ~ On output, it is completely overwritten by the solution $X$. \\
\hline
\mbox{\texttt{ in}}  & {\em Coef\+Mat\+LUP} & \+: The input contiguous square matrix of shape {\ttfamily (nd,nd)} of the same type and kind as {\ttfamily Solution}, representing the LU-\/\+Pivoted (LUP) decomposition of the matrix of coefficients $A$ of the linear system. This matrix is returned by the call to \mbox{\hyperlink{interfaceMatrixLUP__mod_1_1getLUP}{get\+LUP}}. \\
\hline
\mbox{\texttt{ out}}  & {\em Row\+Perm} & \+: The input contiguous array of shape {\ttfamily (nd)} of type {\ttfamily integer} of default kind \mbox{\hyperlink{namespaceConstants__mod_a2bf4d6767c9954d3a495d10ab6b538d6}{IK}} containing the pivot indices. This vector is returned by the call to \mbox{\hyperlink{interfaceMatrixLUP__mod_1_1getLUP}{get\+LUP}}.\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
The dimensions of the input arguments {\ttfamily Solution}, {\ttfamily Coef\+Mat\+LUP}, and {\ttfamily Row\+Perm} must be consistent with each other as described in the above. This condition is only checked for when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1} is set at build time.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}
This routine takes into account the possibility that {\ttfamily Solution} will begin with many zero elements, and is therefore efficient for matrix inversion.
\end{DoxyRemark}
\begin{DoxyNote}{Note}
The input {\ttfamily Coef\+Mat\+LUP} and {\ttfamily Row\+Perm} arguments are not modified by this routine and, can be left in place for successive calls with different right-\/hand sides {\ttfamily Solution}.
\end{DoxyNote}
