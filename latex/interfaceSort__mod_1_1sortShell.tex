Sort the input contiguous {\ttfamily Array} of rank {\ttfamily 1} in ascending order, using the Shell sorting algorithm.  




\doxysubsection{Detailed Description}
Sort the input contiguous {\ttfamily Array} of rank {\ttfamily 1} in ascending order, using the Shell sorting algorithm. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Array} & \+: The contiguous array of rank {\ttfamily 1} of either type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} or type {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ae3573d46486b5f1a0a8d5033a9d12879}{IK8}}, \mbox{\hyperlink{namespaceConstants__mod_acd3c33e94d12f6e7d7c0ee79b2d6280b}{IK16}}, \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, or \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}} whose elements will be sorted in ascending order on return.\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
The size of the input array must not exceed the maximum value representable by the default {\ttfamily integer} kind \mbox{\hyperlink{namespaceConstants__mod_a2bf4d6767c9954d3a495d10ab6b538d6}{IK}}.

This algorithm is provided only for comparison and benchmarking purposes. Its usage for generic applications is {\bfseries{not}} recommended. The \mbox{\hyperlink{interfaceSort__mod_1_1sort}{sort}} algorithm should be the default choice for any sorting in almost all scenarios.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}
This procedure is {\ttfamily pure}.

The use of the Fortran intrinsic {\ttfamily minloc} in this routine appears to lead a slightly better performance than the manual search.
\end{DoxyRemark}
\begin{DoxyNote}{Note}
This algorithm is of order $ N^2$.

To sort the input array in descending order, simply call the output array elements from the end to the beginning or, rewrite the array with {\ttfamily Array = Array( ubound(Array, dim = 1) \+: lbound(Array, dim = 1) \+: -\/1 )}.

A rooter array can be sorted along with a leader array with the help of \mbox{\hyperlink{interfaceSort__mod_1_1sortIndex}{sort\+Index}}.
\end{DoxyNote}
