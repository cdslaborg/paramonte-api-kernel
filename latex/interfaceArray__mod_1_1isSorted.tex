\doxysubsection{Detailed Description}
Return {\ttfamily .true.} if the input array is sorted, either ascending or descending, or all equal, as specified by the user. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Array} & \+: The contiguous array of rank {\ttfamily 1} of either ~ type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} or, ~ type {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ae3573d46486b5f1a0a8d5033a9d12879}{IK8}}, \mbox{\hyperlink{namespaceConstants__mod_acd3c33e94d12f6e7d7c0ee79b2d6280b}{IK16}}, \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, or \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}} or, ~ type assumed-\/length character of default kind or, ~ type \mbox{\hyperlink{structString__mod_1_1String__type}{String\+\_\+type}} or, ~ also, a {\itshape scalar} assumed-\/length character of default kind, ~ whose elements will be checked an all ascending, or all descending order with the possibility elements being equal. \\
\hline

\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily sorted} \+: An output {\ttfamily logical} of default kind that is {\ttfamily .true.} if all elements of the input array are in ascending order, in descending order, or all equal. Otherwise, it is {\ttfamily .false.}.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
If the length of the input {\ttfamily Array} is longer than the largest number that can be represented with an {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, then, the length of array must be provided as the input {\ttfamily integer} argument {\ttfamily len\+Array} of kind \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}}.

The output of this procedure is undefined when the input array has zero length. Furthermore, if the preprocessor flag {\ttfamily CHECK\+\_\+\+ENABLED=1} is set during the build time, an input empty array will halt the program with a call to intrinsic statement {\ttfamily error stop}.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
\begin{DoxyNote}{Note}
If the input optional arguments are {\ttfamily ascending = .false.} and {\ttfamily descending = .false.}, then the output {\ttfamily sorted = .true.} only if all elements of the array are identical.
\end{DoxyNote}
