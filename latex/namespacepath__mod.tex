\begin{DoxyWarning}{Warning}
This code assumes that the input path is a Linux path. Windows paths like {\ttfamily .\textbackslash{}(paramonte)\textbackslash{}paramonte.\+nml} will be horribly treated by this routine as {\ttfamily \textbackslash{}(} also represents a Linux escape character. The result will be {\ttfamily .(paramonte)\textbackslash{}paramonte.\+nml}.

This routine strictly assumes that there is no dangling {\ttfamily \textbackslash{}} in the input Linux path, and if there is, then either it is used to escape the special shell characters, or otherwise, the path is a Windows path.
