Return the normalized sample of shape {\ttfamily (nd,np)} (default) or {\ttfamily (np,nd)} with respect to the input mean vector of length {\ttfamily nd}, where {\ttfamily nd} represents the number of dimensions (data attributes) of the input {\ttfamily Sample} and {\ttfamily np} represents the number of data points in the {\ttfamily Sample}. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Sample} & \+: The array of shape {\ttfamily (nd,np)} or {\ttfamily (np,nd)} containing the sample. \\
\hline
\mbox{\texttt{ in}}  & {\em Mean} & \+: The contiguous vector of mean the input sample. If needed, the input {\ttfamily Mean} can be computed by calling the procedures collectively represented by the generic interface \mbox{\hyperlink{interfaceSample__mod_1_1getMean}{Sample\+\_\+mod\+::get\+Mean()}}. \\
\hline
\mbox{\texttt{ in}}  & {\em dim} & \+: An integer indicating which dimension of the input {\ttfamily Sample} iterates over the individual data. If {\ttfamily dim = 1} or {\ttfamily dim /= 2}, the input {\ttfamily Sample} is assumed to have the shape {\ttfamily (np,nd)}. If {\ttfamily dim = 2}, the input {\ttfamily Sample} is assumed to have the shape {\ttfamily (nd,np)}. ({\bfseries{optional}}, default = {\ttfamily 2}.)\\
\hline
\end{DoxyParams}
\begin{DoxyRemark}{Remarks}
This procedure has the same functionality as \mbox{\hyperlink{namespaceSample__mod_a1b559716e870c15d4ccf9edfdda3ffe7}{Sample\+\_\+mod\+::get\+Normed\+Sample\+\_\+2D()}}. However, unlike {\ttfamily \mbox{\hyperlink{namespaceSample__mod_a1b559716e870c15d4ccf9edfdda3ffe7}{get\+Normed\+Sample\+\_\+2\+D()}}} which is a function and returns the normalized data in a new allocatable object, this {\itshape subroutine} normalizes the input data in place. This is, in general, more efficient and less memory-\/consuming as the cost of destroying the original data.
\end{DoxyRemark}
