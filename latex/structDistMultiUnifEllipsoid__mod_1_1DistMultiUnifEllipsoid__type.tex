\doxysubsection{Detailed Description}
This type creates objects for representing Multi\+Variate Uniform distribution with ellipsoidal support (domain of definition). 

\begin{DoxyRemark}{Remarks}
Unlike other multivariate distributions, the covariance matrix along with its inverse and Cholesky factorization are not components of this type since they already exist as components of the {\ttfamily Support} component of this type, which is an object of type \mbox{\hyperlink{structDomainEllipsoid__mod_1_1DomainEllipsoid__type}{Domain\+Ellipsoid\+\_\+type}}.
\end{DoxyRemark}
