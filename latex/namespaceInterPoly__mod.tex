This is the constructor for objects of type \mbox{\hyperlink{structInterPoly__mod_1_1InterPoly__type}{Inter\+Poly\+\_\+type}}. 

The input {\ttfamily SortedX} and {\ttfamily SortedY} are used to construct the properties of a polynomial interpolation of {\ttfamily degree} degree. Once the interpolation coefficients are computed, user can call the returned object\textquotesingle{}s \mbox{\hyperlink{namespaceInterPoly__mod_a3236e6d69e7c19995b150ce563733874}{predict()}} method to interpolate at arbitrary input query points.
