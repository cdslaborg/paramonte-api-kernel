\begin{DoxyWarning}{Warning}
If {\ttfamily ndiag} is simultaneously present as input argument with {\ttfamily srow} and {\ttfamily scol} input arguments, then the conditions {\ttfamily size(Matrix, dim = 1) \texorpdfstring{$>$}{>}= srow + ndiag -\/ 1} and {\ttfamily size(Matrix, dim = 2) \texorpdfstring{$>$}{>}= scol + ndiag -\/ 1} must hold. These conditions are checked only when the preprocessor flag {\ttfamily CHECK\+\_\+\+ENABLED=1} is set at build time.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
