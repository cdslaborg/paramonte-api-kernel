Assign the contents of the input {\ttfamily string2} to the input {\ttfamily string1}.  




\doxysubsection{Detailed Description}
Assign the contents of the input {\ttfamily string2} to the input {\ttfamily string1}. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em String1} & \+: The input scalar or contiguous array of rank {\ttfamily 1} of type \mbox{\hyperlink{structString__mod_1_1String__type}{String\+\_\+type}} whose contents will be the same as the contents of the input {\ttfamily String2} argument upon return. \\
\hline
\mbox{\texttt{ in}}  & {\em String2} & \+: The input scalar or contiguous array of rank {\ttfamily 1} of type \mbox{\hyperlink{structString__mod_1_1String__type}{String\+\_\+type}} whose contents will be copied to the input {\ttfamily String1} argument upon return.\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
If {\ttfamily String1} is a scalar, then {\ttfamily String2} shall be a scalar. If {\ttfamily String1} is an array, then {\ttfamily String2} shall be either a scalar or an array of the same type and size as {\ttfamily String1}.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
