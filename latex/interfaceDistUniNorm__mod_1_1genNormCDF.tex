Generate the Cumulative Distribution Function (CDF) of the Normal distribution at the given input point.  




\doxysubsection{Detailed Description}
Generate the Cumulative Distribution Function (CDF) of the Normal distribution at the given input point. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Point} & \+: The input scalar or {\ttfamily contiguous} array of rank 1 of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}, representing the point(s) at which the CDF must be computed. \\
\hline
\mbox{\texttt{ in}}  & {\em mean} & \+: The input scalar of the same type and kind as {\ttfamily Point} representing the mean of the distribution ({\bfseries{optional}}, default = {\ttfamily 0.\+\_\+\+RK}). \\
\hline
\mbox{\texttt{ in}}  & {\em inv\+Std} & \+: The input scalar of the same type and kind as {\ttfamily Point} representing the inverse of the standard deviation of the distribution The positivity of the input value is checked only if the library is built with the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. ({\bfseries{optional}}, default = {\ttfamily 1.\+\_\+\+RK}. If present, then the input argument {\ttfamily mean} must be also present).\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily CDF} \+: The output scalar of the same type, kind, and rank as the input {\ttfamily Point} representing the CDF of the Normal distribution at the given input {\ttfamily Point}s.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
This {\ttfamily pure} procedures documented herein become {\ttfamily impure} when the Para\+Monte library is compiled with preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, {\ttfamily pure} procedures are {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
\begin{DoxyNote}{Note}
The fact that the input argument {\ttfamily mean} is required to be present when {\ttfamily std\+Ind} is present has good generic programming and performance reasons (over the alternative of making it an optional input argument when {\ttfamily inv\+Std} is present).
\end{DoxyNote}
