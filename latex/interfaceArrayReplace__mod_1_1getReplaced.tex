\doxysubsection{Detailed Description}
Replace the requested instances of the input {\ttfamily Pattern} with the input {\ttfamily Replacement} in the allocatable input/output {\ttfamily Array}. If a vector of {\ttfamily Instance} is specified, representing the specific instances of Pattern to change, then only those specific instances will be changed.
