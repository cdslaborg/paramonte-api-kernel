Return the cumulative sum of the input {\ttfamily Array}, optionally in the backward direction and optionally, reverse the output cumulative sum upon return.  




\doxysubsection{Detailed Description}
Return the cumulative sum of the input {\ttfamily Array}, optionally in the backward direction and optionally, reverse the output cumulative sum upon return. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Array} & \+: The input contiguous array of shape {\ttfamily (\+:)} of either type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} or type {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ae3573d46486b5f1a0a8d5033a9d12879}{IK8}}, \mbox{\hyperlink{namespaceConstants__mod_acd3c33e94d12f6e7d7c0ee79b2d6280b}{IK16}}, \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, or \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}} whose cumulative sum will have to be computed. \\
\hline
\mbox{\texttt{ out}}  & {\em Cum\+Sum} & \+: The output contiguous array of the same size, shape, type, and kind as the input {\ttfamily Array} containing the cumulative sum of {\ttfamily Array} in the specified direction. \\
\hline
\mbox{\texttt{ in}}  & {\em backward} & \+: The input scalar of type {\ttfamily logical} of default kind. If {\ttfamily .true.}, the output {\ttfamily Cum\+Sum} will be computed from the last element to the first element of the input {\ttfamily Array} ({\bfseries{optional}}, default = {\ttfamily .false.}). \\
\hline
\mbox{\texttt{ in}}  & {\em reversed} & \+: The input scalar of type {\ttfamily logical} of default kind. If {\ttfamily .true.}, the output array {\ttfamily Cum\+Sum} will be reversed upon return, such that its last element becomes the first ({\bfseries{optional}}, default = {\ttfamily .false.}. If present, then the optional input argument {\ttfamily backward} must be also present).\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
The size of the {\ttfamily Array} and {\ttfamily Cumsum} arguments must be the same. This condition is only checked when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1} is set.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
