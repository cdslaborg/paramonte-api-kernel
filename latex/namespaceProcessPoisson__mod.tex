{\footnotesize\ttfamily subroutine Process\+Poisson\+\_\+mod\+::get\+Dist\+Sorted\+Exp\+Diff\+Weight\+KS2 (\begin{DoxyParamCaption}\item[{integer(IK)}]{Sorted\+Index,  }\item[{real(RK)}]{Dist\+Sorted\+Exp\+Diff,  }\item[{real(RK)}]{log\+Vol\+Unit\+Ball,  }\item[{real(RK)}]{Prob\+KS,  }\item[{real(RK)}]{Stat\+KS,  }\item[{real(RK)}]{Lambda,  }\item[{real(RK), intent(out), optional}]{alpha,  }\item[{real(RK), intent(out), optional}]{beta,  }\item[{type(\mbox{\hyperlink{structErr__mod_1_1Err__type}{Err\+\_\+type}})}]{Err }\end{DoxyParamCaption})}



Return the weights (the KS-\/test probabilities) of the the input {\ttfamily Dist\+Sorted\+Exp\+Diff} resulting from the same exponential distribution. Optionally, also return the vector of KS-\/statistic corresponding to each element of {\ttfamily Dist\+Sorted\+Exp\+Diff}. The output {\ttfamily Prob\+KS} can be used in the weighted likelihood function of the exponential distribution to robustly infer the parameters of the Gamma posterior of the rate parameter of the exponential distribution from which the input {\ttfamily Dist\+Sorted\+Exp\+Diff} results. , or if {\ttfamily log\+Vol\+Unit\+Ball} is present among the input arguments, return the parameters of the posterior distribution of density of the PPP that yielded the input {\ttfamily Dist\+Sorted\+Exp\+Diff}. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Sorted\+Index} & \+: The input contiguous array of indices of {\ttfamily Dist\+Sorted\+Exp\+Diff} whose weights will be directly computed. The trivial case is to pass a linear range of indices {\ttfamily \mbox{[}(i, i = 1, np)\mbox{]}}. The longer the length of {\ttfamily Index}, the more costly the computations will be, especially if {\ttfamily Index} contains indices of the later elements of {\ttfamily Dist\+Sorted\+Exp\+Diff}. \\
\hline
\mbox{\texttt{ in}}  & {\em Dist\+Sorted\+Exp\+Diff} & \+: The input contiguous array of size {\ttfamily (np)} of exponentiated-\/distance differences. This array is returned by \mbox{\hyperlink{namespaceGeometry__mod_aa4755e0e280bb7e8a213e8b85663cf7f}{get\+Dist\+Sorted\+Exp\+Diff()}}. \\
\hline
\mbox{\texttt{ out}}  & {\em Prob\+KS} & \+: The output contiguous array of size {\ttfamily (np)} of KS two-\/sample probabilities corresponding to each element in {\ttfamily Dist\+Sorted\+Exp\+Diff} ({\bfseries{optional}}). \\
\hline
\mbox{\texttt{ out}}  & {\em Stat\+KS} & \+: The output contiguous array of size {\ttfamily (np)} of KS two-\/sample distance metric corresponding to each element in {\ttfamily Dist\+Sorted\+Exp\+Diff} ({\bfseries{optional}}). \\
\hline
\mbox{\texttt{ out}}  & {\em Lambda} & \+: The output contiguous array of size {\ttfamily (np)} of the sample-\/size-\/adjusted KS two-\/sample test statistic corresponding to each element in {\ttfamily Dist\+Sorted\+Exp\+Diff} ({\bfseries{optional}}). \\
\hline
\mbox{\texttt{ out}}  & {\em alpha} & \+: The shape parameter of the Gamma posterior probability density function (PDF) of the density ({\bfseries{optional}}). \\
\hline
\mbox{\texttt{ out}}  & {\em beta} & \+: The scale parameter of the Gamma posterior PDF function of the density ({\bfseries{optional}}). \\
\hline
\mbox{\texttt{ out}}  & {\em Err} & \+: An object of class \mbox{\hyperlink{structErr__mod_1_1Err__type}{Err\+\_\+mod\+::Err\+\_\+type}} containing error information. \\
\hline
\mbox{\texttt{ in}}  & {\em log\+Vol\+Unit\+Ball} & \+: The natural logarithm of the volume of a unit ball in the dimension of interest {\ttfamily nd}. This quantity is returned by \mbox{\hyperlink{interfaceGeometryEllipsoid__mod_1_1genLogVolUnitBall}{gen\+Log\+Vol\+Unit\+Ball()}}. ({\bfseries{optional}}, if provided, then {\ttfamily beta} will be multiplied with {\ttfamily exp(log\+Vol\+Unit\+Ball)} before return. This is equivalent to the {\ttfamily beta} parameter of the original PPP leading to the input {\ttfamily Dist\+Sorted\+Exp\+Diff}. If {\ttfamily log\+Vol\+Unit\+Ball} is missing among the input arguments, then the return beta will describe the posterior density of the rate parameter of the exponential model that describes the input {\ttfamily Dist\+Sorted\+Exp\+Diff}.\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
On input, the vector {\ttfamily Sorted\+Index} must be sorted in {\bfseries{ascending order}} with {\bfseries{unique}} elements.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}
This method relies on minimizing the KS-\/test statistic. Specifically, input differences vector of exponentiated distances is passed to the KS algorithm iteratively with larger and larger sample sizes to compute the disparity weights of the points. Then, these weights are used to construct the weighted Gamma likelihood function of the points.
\end{DoxyRemark}
