{\footnotesize\ttfamily pure subroutine Hypo\+Test\+\_\+mod\+::get\+Sample\+Disparity (\begin{DoxyParamCaption}\item[{integer(IK)}]{Index,  }\item[{real(RK)}]{Sorted\+Sample,  }\item[{real(RK)}]{Prob\+KS,  }\item[{real(RK)}]{Stat\+KS,  }\item[{real(RK)}]{Lambda,  }\item[{type(\mbox{\hyperlink{structErr__mod_1_1Err__type}{Err\+\_\+type}})}]{Err }\end{DoxyParamCaption})}



Return the weights of each of the points in the input vector of {\ttfamily Point((2\+:np)} as {\ttfamily Prob\+KS}. These KS two-\/sample test probabilities are obtained by dividing the input sample into approximately equal sizes and computing the quantities of interest. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Index} & \+: The index of the points in the input {\ttfamily Point} for which the weights will be computed. The smaller the length of this vector, the less it will take to return the results. In return, the corresponding probabilities and statistics for the missing points can be approximated via linear interpolation from the existing neighbors. \\
\hline
\mbox{\texttt{ in}}  & {\em Sorted\+Sample} & \+: The input array of points of shape {\ttfamily (np)} sorted in order of {\bfseries{relevance (unbiasedness)}}. \\
\hline
\mbox{\texttt{ out}}  & {\em Prob\+KS} & \+: The output vector of the same length as {\ttfamily Index} containing the two-\/sample KS test probabilities (the wieghts). \\
\hline
\mbox{\texttt{ out}}  & {\em Stat\+KS} & \+: The output vector of the same length as {\ttfamily Index} containing the two-\/sample KS test statistics. \\
\hline
\mbox{\texttt{ out}}  & {\em lambda} & \+: The output vector of the same length as {\ttfamily Index} containing the sample-\/size-\/adjusted two-\/sample KS test statistics. \\
\hline
\mbox{\texttt{ out}}  & {\em Err} & \+: An object of class \mbox{\hyperlink{structErr__mod_1_1Err__type}{Err\+\_\+type}} containing error information.\\
\hline
\end{DoxyParams}
\begin{DoxyRemark}{Remarks}
The output vector of probabilities and the corresponding KS statistics are not unreliable for any value of {\ttfamily Index} that is less than 8, since the corresponding effective sample sizes {\ttfamily N\+\_\+e} are less than four.

For any value {\ttfamily Index} that is unity, the effective sample size will be 0\+: $ N_e = \frac{N_1 \times N_2}{N_1 + N_2} $. As such, for the sake of comprehensiveness, its KS probability is assigned a value of {\ttfamily 1} and KS statistic of {\ttfamily 0}.
\end{DoxyRemark}
