\doxysubsection{Detailed Description}
Return a matrix resulting from the multiplication of the input {\ttfamily facupp} with the upper triangle of the input square matrix and {\ttfamily faclow} with the lower triangle of the input matrix and {\ttfamily facdia} with the diagonal elements of the input matrix. 

This function is a shortcut to avoid writing loops for multiplying the upper and lower triangles and diagonal of a matrix by some given factors. While writing the loop is simple, since it occurs frequently, this function provides a cleaner solution.
