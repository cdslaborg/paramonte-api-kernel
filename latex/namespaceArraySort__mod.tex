\doxysubsection{Detailed Description}
This module contains procedures for sorting arrays and merging sorted arrays. 

\begin{DoxyNote}{Note}
Recommended routines for sorting arrays\+:
\begin{DoxyItemize}
\item The procedures under the generic interface \mbox{\hyperlink{interfaceArraySort__mod_1_1sort}{sort}} are among the fastest sorting algorithms particularly for fully random input arrays.
\end{DoxyItemize}
\end{DoxyNote}
