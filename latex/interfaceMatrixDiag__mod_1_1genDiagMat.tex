\doxysubsection{Detailed Description}
Generate a diagonal matrix of shape {\ttfamily (nrow,ncol)} with its diagonal elements set to the input {\ttfamily Diag}. All off-\/diagonal elements are set to zero.

\begin{DoxyWarning}{Warning}
If the input {\ttfamily Diag} is a contiguous array and {\ttfamily nrow} and {\ttfamily ncol} are also present as input arguments, then the conditions {\ttfamily nrow \texorpdfstring{$>$}{>}= srow + size(\+Diag) -\/ 1} and {\ttfamily ncol \texorpdfstring{$>$}{>}= scol + size(\+Diag) -\/ 1} must hold. These conditions are checked only when the preprocessor flag {\ttfamily CHECK\+\_\+\+ENABLED=1} is set at build time.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
