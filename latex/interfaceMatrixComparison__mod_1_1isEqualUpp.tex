\doxysubsection{Detailed Description}
Generate a logical value of {\ttfamily .true.} if the upper-\/triangle ~\newline
 elements of the two input square matrices are equal within the given input threshold. 

This function interface is a shortcut to avoid loops for comparing upper triangles of two matrices. While writing the loop is simple, this function provides a cleaner solution.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Matrix1} & \+: The first input contiguous square matrix of shape {\ttfamily (ndim,ndim)} of type {\ttfamily real} and kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}. \\
\hline
\mbox{\texttt{ in}}  & {\em Matrix2} & \+: The second input contiguous square matrix of the same shape, type, and kind as {\ttfamily Matrix1}. \\
\hline
\mbox{\texttt{ in}}  & {\em tol} & \+: The threshold below which the corresponding elements of the two matrices are assumed to be equal.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily is\+Equal} \+: The output {\ttfamily logical} of default kind \mbox{\hyperlink{namespaceConstants__mod_a68ac316f10b57be996670bc7c0abed9d}{LK}}, indicating if the all elements of the upper-\/triangles of the two input matrices are equal within the specified tolerance {\ttfamily tol}.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
The input matrices must be square. This condition is checked only when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED} evaluates to true.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
