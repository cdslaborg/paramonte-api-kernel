\hline

\hline

\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily Lower\+Copy} \+: The lower copy of the input lower-\/triangular matrix.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
Note that only the values in the lower-\/triangle of the input {\ttfamily Lower} matrix are copied to the output lower-\/triangle matrix. Furthermore, the upper-\/triangle of the output {\ttfamily Lower\+Copy} matrix is NOT initialized to any specific value, such as {\ttfamily 0}.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}
This procedure is developed for the purpose of avoiding the unnecessary copy of redundant data in the upper-\/triangles of matrices. An example use case is when a copy of the lower-\/triangle of a Cholesky factorization has to be made.
