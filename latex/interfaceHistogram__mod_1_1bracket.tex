Return the lower and upper indices of the elements of the ascending-\/sorted input vector {\ttfamily Ascending\+Vector} that sandwich the input {\ttfamily value}. The search method is linear. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Ascending\+Vector} & \+: The input contiguous vector of values. \\
\hline
\mbox{\texttt{ in}}  & {\em value} & \+: The target value to search for in the input vector. \\
\hline
\mbox{\texttt{ in}}  & {\em lower\+Index} & \+: The lower index of the element of {\ttfamily Ascending\+Vector} that is the largest element smaller than or equal to {\ttfamily value}. \\
\hline
\mbox{\texttt{ in}}  & {\em upper\+Index} & \+: The lower index of the element of {\ttfamily Ascending\+Vector} that is the smallest element larger than {\ttfamily value}.\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
If the input {\ttfamily value} is smaller than all elements of {\ttfamily Ascending\+Vector}, then {\ttfamily lower\+Index = 0} and {\ttfamily upper\+Index = 1}.

If the input {\ttfamily value} is larger than all elements of {\ttfamily Ascending\+Vector}, then {\ttfamily lower\+Index = size(\+Ascending\+Vector)} and {\ttfamily upper\+Index = lower\+Index + 1}.
