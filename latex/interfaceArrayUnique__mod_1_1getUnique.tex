\doxysubsection{Detailed Description}
Return a vector of unique values in the input array {\bfseries{in place of}} the array itself. The uniqueness of the values can be optionally determined by the user by specifying the external input function that checks the equivalence of a pair of elements of the input {\ttfamily Array}. Furthermore, optionally return the count of each of the unique values in the input {\ttfamily Array}, as well as the indices of occurrences of each of the unique values.

\begin{DoxyWarning}{Warning}

Note that in Fortran, trailing blanks are ignored in character comparison, that is, {\ttfamily \char`\"{}\+Fortran\char`\"{} == \char`\"{}\+Fortran \char`\"{}} yields {\ttfamily .true.}.
