\doxysubsection{Detailed Description}
Copy the upper-\/triangle (excluding diagonals) of the input {\ttfamily from} square matrix to the upper-\/triangle of the input matrix {\ttfamily to}. 

This subroutine interface is a shortcut to avoid loops for copying the upper triangle of a matrix to another existing one. While writing the loop is simple, since it occurs frequently, these subroutines provide a cleaner solution.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em from} & \+: The contiguous square matrix whose upper-\/triangle will have to be copied into the destination matrix. All elements of {\ttfamily from} shall be all reals of either {\ttfamily RK32}, {\ttfamily RK64}, {\ttfamily RK128} kinds. \\
\hline
\mbox{\texttt{ in,out}}  & {\em to} & \+: The destination matrix of the same type and kind as {\ttfamily from} to which the upper-\/triangle of {\ttfamily from} will be copied.\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
The input matrices {\ttfamily from} and {\ttfamily to} must be square. This condition is checked only when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED} evaluates to true.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in release build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
