\begin{DoxyWarning}{Warning}
If any error occurs during the ellipsoid construction, the first element of the {\ttfamily Cho\+Dia} component of the output {\ttfamily Ellipsoid} object will be set to a negative number on return. This is to preserve the purity of the function and to avoid the need for error handling.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}

\end{DoxyRemark}
