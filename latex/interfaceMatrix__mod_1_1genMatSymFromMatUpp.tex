Generate and return a symmetric matrix from the input upper-\/triangular square matrix.  




\doxysubsection{Detailed Description}
Generate and return a symmetric matrix from the input upper-\/triangular square matrix. 

This function is a handy shortcut for symmetrizing upper-\/triangular square matrices. Optionally, it also sets the diagonal elements of the output generated matrix to the input {\ttfamily Dia} vector, if provided as input.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Mat\+Upp} & \+: The square matrix whose elements shall be all reals of either \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} kinds. \\
\hline
\mbox{\texttt{ in}}  & {\em Dia} & \+: The diagonal elements of the output matrix. This optional input argument is particularly useful when symmetrizing the upper triangle of a Cholesky factorization. ({\bfseries{optional}}, The default will be the diagonal elements of {\ttfamily Mat\+Upp}).\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily Mat\+Sym} \+: The symmetric output real matrix of the same real-\/kind as the input {\ttfamily Mat\+Upp}. The upper triangle of {\ttfamily Mat\+Sym} is the same as that of the input {\ttfamily Mat\+Upp} and its lower triangle is the transpose of the upper triangle of {\ttfamily Mat\+Upp} and its diagonal elements are either taken from the diagonals of the input {\ttfamily Mat\+Upp} if the input argument {\ttfamily Dia} is missing, or else, it is set to the input vector {\ttfamily Dia}.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
The input matrix must be square. This condition is checked only when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED} evaluates to true.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
