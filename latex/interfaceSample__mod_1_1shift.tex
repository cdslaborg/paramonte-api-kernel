Return the shifted sample of shape {\ttfamily (nd,np)} (default) or {\ttfamily (np,nd)} by the input {\ttfamily Amount}, a contiguous vector of length {\ttfamily nd}, where {\ttfamily nd} represents the number of dimensions (data attributes) of the input {\ttfamily Sample} and {\ttfamily np} represents the number of data points in the {\ttfamily Sample}. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Sample} & \+: The array of shape {\ttfamily (nd,np)} or {\ttfamily (np,nd)} containing the sample. \\
\hline
\mbox{\texttt{ in}}  & {\em Amount} & \+: The contiguous vector of the amount by which the sample must be shifted. For example, if the input {\ttfamily Sample} is to be shifted such that its new mean is the origin, then {\ttfamily Amount = -\/mean(\+Sample)}. If needed, the current mean can be computed by calling the procedures collectively represented by the generic interface \mbox{\hyperlink{interfaceSample__mod_1_1genMean}{Sample\+\_\+mod\+::gen\+Mean()}}. \\
\hline
\mbox{\texttt{ in}}  & {\em dim} & \+: An integer indicating which dimension of the input {\ttfamily Sample} iterates over the individual data. If {\ttfamily dim = 1} or {\ttfamily dim /= 2}, the input {\ttfamily Sample} is assumed to have the shape {\ttfamily (np,nd)}. If {\ttfamily dim = 2}, the input {\ttfamily Sample} is assumed to have the shape {\ttfamily (nd,np)}. ({\bfseries{optional}}, default = {\ttfamily 2}.)\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in release build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds. 
\end{DoxyWarning}

This is, in general, more efficient and less memory-\/consuming as the cost of destroying the original data.
