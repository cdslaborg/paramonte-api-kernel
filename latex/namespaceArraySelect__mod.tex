This benchmark represents the worst-\/case scenario. Note that this benchmark is not relevant to selecting substrings to Fortran strings.

\begin{DoxyEnumerate}
\item The procedures under the generic interface \mbox{\hyperlink{interfaceArraySelect__mod_1_1genSelected}{gen\+Selected}} are functions while the procedures under the generic interface \mbox{\hyperlink{interfaceArraySelect__mod_1_1select}{select}} are subroutines. From the benchmark results, it appears that the functional interface performs slightly less efficiently than the subroutine interface. The sole apparent reason for this performance loss seems to be the extra copy of the result to the allocatable {\ttfamily Array\+New} on return from the function. Note that this benchmark does not even include the cost of repeated reallcations, that is, the allocation of {\ttfamily Array\+New} happen only once in all tests.
\item Note that this benchmark considers the worst-\/case scenario where {\ttfamily Selection} must be selected at all positions of the input {\ttfamily Array}.
\end{DoxyEnumerate}
