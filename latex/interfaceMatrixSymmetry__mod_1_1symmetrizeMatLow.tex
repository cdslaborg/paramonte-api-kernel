\doxysubsection{Detailed Description}
Return the input square lower-\/triangular matrix by symmetrizing it. Optionally, also overwrite the diagonal elements of the matrix with the input diagonal vector {\ttfamily Dia}. 

This subroutine is a handy shortcut for symmetrizing lower-\/triangular square matrices {\bfseries{in-\/place}}, that is, by modifying the input matrix. Optionally, it also sets the diagonal elements of the output generated matrix to the input {\ttfamily Dia} vector, if provided as input.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Mat\+Upp} & \+: The square matrix whose elements shall be all reals of either \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} kinds. On output, the upper triangle of {\ttfamily Mat\+Upp} is overwritten with the transpose of its lower tirangle. \\
\hline
\mbox{\texttt{ in}}  & {\em Dia} & \+: The diagonal elements of the output matrix. It must be of the same type and kind as the input {\ttfamily Mat\+Upp}. If present, it will overwrite the existing diagonal elements of the input matrix. This optional input argument is particularly useful when symmetrizing the lower triangle of a Cholesky factorization. ({\bfseries{optional}}, If missing, the diagonal elements of the input {\ttfamily Mat\+Upp} will remain intact).\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
The input matrix must be square. This condition is checked only when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED} evaluates to true.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
