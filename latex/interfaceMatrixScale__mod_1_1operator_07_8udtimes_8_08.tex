Return a matrix resulting from the multiplication of the input factor with the upper triangle of the input square matrix.  




\doxysubsection{Detailed Description}
Return a matrix resulting from the multiplication of the input factor with the upper triangle of the input square matrix. 

This function and its operator interface is a shortcut to avoid loops for multiplying only the upper triangle of a matrix by a factor. While writing the loop is simple, since it occurs frequently, this function provides a cleaner solution.
