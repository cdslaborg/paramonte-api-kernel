\doxysubsection{Detailed Description}
Return the lower-\/triangle of the Cholesky factorization $L$ of the symmetric positive-\/definite matrix represented by the upper-\/triangle of the input argument $\texttt{ChoLowMatUpp} = L.L^T$. On input, the upper triangle and diagonal of {\ttfamily Cho\+Low\+Mat\+Upp} must be given, which remains intact on output. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em nd} & \+: The input integer of default kind \mbox{\hyperlink{namespaceConstants__mod_a2bf4d6767c9954d3a495d10ab6b538d6}{IK}} representing the rank of the input square matrix {\ttfamily (nd,nd)} ({\bfseries{optional}}, must be present if and only if the output argument {\ttfamily failed} is missing). \\
\hline
\mbox{\texttt{ in,out}}  & {\em Cho\+Low\+Mat\+Upp} & \+: The input square symmetric positive-\/definite (covariance) matrix of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}. Only the upper half and triangle of the matrix are needed as input. On output, the lower half of the matrix will be rewritten with the lower triangle of the Cholesky Factorization. \\
\hline
\mbox{\texttt{ out}}  & {\em Cho\+Dia} & \+: The output vector of the same type and kind as {\ttfamily Cho\+Low\+Mat\+Upp} containing the diagonal elements of the lower triangle of the Cholesky factorization. \\
\hline
\mbox{\texttt{ out}}  & {\em failed} & \+: The output {\ttfamily logical} of default kind that is {\ttfamily .true.} if the Cholesky factorization fails ({\bfseries{optional}}, must be present if and only if the input argument {\ttfamily nd} is missing).\\
\hline
\end{DoxyParams}

\begin{DoxyWarning}{Warning}
The interface with explicit-\/shape dummy arguments is not recommended for general usage. It is available only for internal benchmarking purposes. However, the following rules hold if they are used. If Cholesky factorization fails, {\ttfamily Cho\+Dia(1)} will be set to {\ttfamily -\/1} to indicate an error on return. This is necessary to maintain the purity of the procedure. However, if the library is built with the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED} set to true, then the occurrence of an error will halt the program by executing the {\ttfamily error stop} statement.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}
If {\ttfamily nd = 1}, {\ttfamily Cho\+Low\+Mat\+Upp} will not be touched, only {\ttfamily sqrt(\+Cho\+Low\+Mat\+Upp)} will be written to the output {\ttfamily Cho\+Dia}.
\end{DoxyRemark}
