\doxysubsection{Detailed Description}
This is the class for generating instances of grid-\/like Multivariate Normal Distribution mixtures. 

\begin{DoxyRemark}{Remarks}
This class is particularly useful for benchmarking different sampling routines of the Para\+Monte library. 
\end{DoxyRemark}
