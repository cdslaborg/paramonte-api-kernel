\item {\ttfamily TimerCounttotal} \+: The total counts since the object creation or since the last {\ttfamily tic()} call.
\item {\ttfamily TimerCountstop} \+: The current count as inferred from the Fortran intrinsic procedure {\ttfamily system\+\_\+clock()}.
\item {\ttfamily TimerTimerdelta} \+: The total time in seconds since the last {\ttfamily toc()} call.
\item {\ttfamily TimerTimertotal} \+: The total time in seconds since the object creation or since the last {\ttfamily tic()} call.
