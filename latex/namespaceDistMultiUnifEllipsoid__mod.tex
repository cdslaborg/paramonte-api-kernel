Construct an object of type \mbox{\hyperlink{structDistMultiUnifEllipsoid__mod_1_1DistMultiUnifEllipsoid__type}{Dist\+Multi\+Unif\+Ellipsoid\+\_\+type}} containing the properties of the specified Multi\+Variate Uniform (MVU) distribution in {\ttfamily ndim} dimensions. 

An ellipsoidal MVU distribution is specified by the mean of the distribution (or equivalently, the center of its support) as well as its representative matrix. The representative matrix is the matrix $A$ such that for any point $X$ within the support of the distribution, the following relationship holds,

\begin{eqnarray*} X^T ~ A^{-1} ~ X \leq 1 ~, \end{eqnarray*}

In such a case, the covariance of the ellipsoidal distribution is given by

\begin{eqnarray*} \text{CovMat} = \frac{A}{p + 2} ~, \end{eqnarray*}

\begin{DoxyRemark}{Remarks}
If not readily available, an input upper-\/triangle covariance matrix can be constructed by calling gen\+Cov\+Upp\+From\+Cor\+Upp(). Furthermore, the covariance matrix can be converted to a representative matrix by multiplying each element of the covariance matrix with {\ttfamily (ndim + 2)} where {\ttfamily ndim} is the number of dimensions of the domain of the distribution.

This routine computes the inverse of the input covariance matrix. As such, if an error occurs, the first element of {\ttfamily selfCho\+Low} ({\ttfamily selfCho\+Low(1,1)}) will be set to a negative number. Additionally, if the input argument {\ttfamily Err} is present, an error message will be returned in {\ttfamily Errmsg} and {\ttfamily Erroccurred = .true.}.

By default, when the input argument {\ttfamily Rep\+Upp} is missing, the returned ellipsoidal MVU distribution will have an identity representative matrix, meaning that the covariance matrix of the distribution is {\bfseries{not}} identity. To generate a distribution object with identity covariance matrix, pass an identity matrix for {\ttfamily Rep\+Upp} input argument and set the input argument {\ttfamily is\+Cov\+Upp = .true.}.
\end{DoxyRemark}
