Generate and return the variance of the input array of shape {\ttfamily (np)} or {\ttfamily (nd,np)} or {\ttfamily (np,nd)} where {\ttfamily nd} is the number of data dimensions (the number of data attributes) and {\ttfamily np} is the number of data points.  




\doxysubsection{Detailed Description}
Generate and return the variance of the input array of shape {\ttfamily (np)} or {\ttfamily (nd,np)} or {\ttfamily (np,nd)} where {\ttfamily nd} is the number of data dimensions (the number of data attributes) and {\ttfamily np} is the number of data points. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em Sample} & \+: The input contiguous array of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} of shape {\ttfamily (np)}, {\ttfamily (nd,np)}, or {\ttfamily (np,nd)} containing the sample. If {\ttfamily Sample} is a 2D array, then the direction along which variance is computed is dictated by the optional input argument {\ttfamily dim}. \\
\hline
\mbox{\texttt{ in}}  & {\em Weight} & \+: The contiguous vector of shape {\ttfamily (np)} of either the same type and kind as the input {\ttfamily Sample} or type {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ae3573d46486b5f1a0a8d5033a9d12879}{IK8}}, \mbox{\hyperlink{namespaceConstants__mod_acd3c33e94d12f6e7d7c0ee79b2d6280b}{IK16}}, \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, or \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}}, containing the corresponding weight of each data points in {\ttfamily Sample} ({\bfseries{optional}}, default = a vector of ones). \\
\hline
\mbox{\texttt{ in}}  & {\em Mean} & \+: The input scalar or contiguous vector of shape {\ttfamily (nd)} of the same type and kind as the input {\ttfamily Sample} containing the {\ttfamily Sample} mean along the (optionally) specified dimension {\ttfamily dim}. If the input {\ttfamily Sample} is a 1D array, then {\ttfamily Mean} must be a scalar. Otherwise, if {\ttfamily Mean} is a 2D array, then {\ttfamily Mean} must be a vector whose size is the same as the size of at least one of the dimensions of {\ttfamily Sample}. ({\bfseries{optional}}. If missing, then the input argument {\ttfamily shifted} must be present indicating whether the input {\ttfamily Sample} is already centered at the origin or it has to be shifted to the origin by the procedure). \\
\hline
\mbox{\texttt{ in}}  & {\em shifted} & \+: The input {\ttfamily logical} of default kind indicating whether the input {\ttfamily Sample} is already centered at the origin or the it has to be shifted to the origin by the procedure ({\bfseries{optional}}. If missing, then the input argument {\ttfamily Mean} must be present). \\
\hline
\mbox{\texttt{ in}}  & {\em biased} & \+: The input {\ttfamily logical} of default kind indicating whether the output variance should be corrected for small sample-\/size bias. Set this argument to {\ttfamily .false.} to avoid biased variance computation, in particular, when the sample size {\ttfamily np} is small. \\
\hline
\mbox{\texttt{ in}}  & {\em dim} & \+: An integer of default kind \mbox{\hyperlink{namespaceConstants__mod_a2bf4d6767c9954d3a495d10ab6b538d6}{IK}} indicating which dimension of the input {\ttfamily Sample} iterates over the individual data points. If {\ttfamily dim = 1} or {\ttfamily dim /= 2}, the input {\ttfamily Sample} is assumed to have the shape {\ttfamily (np,nd)}. If {\ttfamily dim = 2}, the input {\ttfamily Sample} is assumed to have the shape {\ttfamily (nd,np)} ({\bfseries{optional}}, default = {\ttfamily 2}. {\bfseries{This input argument is available only if the input {\ttfamily Sample} is a 2D array}}.).\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily variance} \+: The output variance of the input sample of the same type and kind as the input {\ttfamily Sample}. It is a scalar only if the input {\ttfamily Sample} is a 1D array. Otherwise, it is an allocatable array of shape {\ttfamily (nd)}.
\end{DoxyReturn}
