\doxysubsection{Detailed Description}
This is the base class for creating low-\/level benchmark objects. 

\begin{DoxyNote}{Note}
Although it is possible, this type is not meant to be directly used for benchmarking. Use instead the child class \mbox{\hyperlink{structBench__mod_1_1Bench__type}{Bench\+\_\+type}} which greatly simplified benchmarking.
