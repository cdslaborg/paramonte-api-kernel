\begin{DoxyWarning}{Warning}
Failure to provide the input optional argument {\ttfamily chain\+Size} will lead to the entire chain file being read into the chain object. This could potentially be catastrophic if the chain size is larger than the available virtual memory. At best, it will result in an IO twice slower than possible, since the file will have to be read twice.

When {\ttfamily chain\+File\+Form = \char`\"{}binary\char`\"{}}, {\ttfamily len\+Header} and {\ttfamily delimiter} must be also provided.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}

\end{DoxyRemark}
