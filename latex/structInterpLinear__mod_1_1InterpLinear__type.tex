The input {\ttfamily SortedX} is the vector of nodes of the linear interpolation, whose elements are sorted in ascending order. The input {\ttfamily SortedY} is the corresponding function values at the given ascending-\/sorted interpolation nodes {\ttfamily SortedY}. are used to construct the properties of a linear interpolation of {\ttfamily degree} degree. The linear interpolation implemented in this constructor is based on the Lagrange classical formula for linear interpolation. Suppose an input query point $x$ falls between two nodes $x_i$ and $x_{i+1}$ with the corresponding function values $y_i$ and $y_{i+1}$ and we wish to estimate the corresponding interpolated value $y(x)$, which can be computed as,

\begin{equation*} y(x) = \frac {x - x_{i+1}} {x_i - x_{i+1}} y_i + \frac {x - x_{i}} {x_{i+1} - x_{i}} y_{i+1} \end{equation*}

Since there are {\ttfamily num\+Node} interpolation nodes, there exist {\ttfamily ninterp = num\+Node -\/ 1} such equations as in the above. To expedite future interpolations based on the input nodes, the two fixed coefficients in the above equation are stored in the {\ttfamily Coef} component on the object for each equation corresponding to a linear interpolation between two adjacet points.

Once the interpolation coefficients are computed, user can call the \mbox{\hyperlink{structInterpLinear__mod_1_1InterpLinear__type_a92adc229bacdf8ff90021aaec0823f8c}{predict()}} method of the returned object to interpolate at arbitrary input query points.
