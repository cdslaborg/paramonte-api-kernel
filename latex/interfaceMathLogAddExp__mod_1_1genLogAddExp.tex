\begin{DoxyWarning}{Warning}
As the argument names suggest, the value of the input argument {\ttfamily larger} must be larger than the the value of the input argument {\ttfamily smaller}. When the input arguments are of type {\ttfamily complex}, this condition must hold for the {\ttfamily real} components of the numbers. The validity of this condition is only checked when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1} is set at build time.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
