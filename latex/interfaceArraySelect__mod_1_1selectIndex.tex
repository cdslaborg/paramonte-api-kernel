\doxysubsection{Detailed Description}
Return the {\ttfamily rank}th smallest (or ordered) value in the input array by first sorting its elements (optionally only between the specified indices {\ttfamily \mbox{[}left,right\mbox{]}}). By default, the array will be sorted in ascending order, unless the user-\/specific {\ttfamily is\+Sorted()} external function is supplied to define a custom sorting criterion.

 This user-\/defined equivalence check is extremely useful where a user-\/defined sorting criterion other than simple ascending order is needed, for example, when the case-\/sensitivity of an input string or array of strings is irrelevant or when sorting of the absolute values matters excluding the signs of the numbers, or when descending order is desired. \begin{DoxyWarning}{Warning}
If {\ttfamily left} is present as an input argument, its value must be larger than {\ttfamily 0} and less than {\ttfamily right} or its default value. Furthermore, the specified {\ttfamily rank} must be larger than or equal to the specified value for {\ttfamily left}. These conditions are only checked for when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLD=1} is set when building the Para\+Monte library.

If {\ttfamily right} is present as an input argument, its value must be less than or equal to the length of {\ttfamily Array} and larger than {\ttfamily right} or its default value. Furthermore, the specified {\ttfamily rank} must be smaller than or equal to the specified value for {\ttfamily right}. These conditions are only checked for when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLD=1} is set when building the Para\+Monte library.

The functions under this generic interface are {\ttfamily impure} when the {\ttfamily external} input argument {\ttfamily is\+Sorted()} is {\ttfamily present}.

This {\ttfamily pure} procedures documented herein become {\ttfamily impure} when the Para\+Monte library is compiled with preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, {\ttfamily pure} procedures are {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
