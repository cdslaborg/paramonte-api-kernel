Solve the set of {\ttfamily nd} linear equations of the form,.  




\doxysubsection{Detailed Description}
Solve the set of {\ttfamily nd} linear equations of the form,. 

\begin{equation} A \cdot X = B \end{equation} where $A$ is the coefficient matrix represented by its Cholesky upper-\/triangular decomposition matrix {\ttfamily Coef\+Mat\+Cho\+Upp} and its diagonals {\ttfamily Coef\+Mat\+Cho\+Dia}, $X$ is the output solution, and $B$ is the right-\/hand-\/side vector of constants.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Solution} & \+: The output contiguous vector of size {\ttfamily nd} of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}, containing the solution $X$. If the input argument {\ttfamily Right\+Hand} is missing, then {\ttfamily Solution} also represents the right-\/hand-\/side $B$ in the system of linear equations to be solved on input. On output, it will be completely overwritten with the solution. \\
\hline
\mbox{\texttt{ in}}  & {\em Coef\+Mat\+Cho\+Upp} & \+: The input contiguous square matrix of shape {\ttfamily (nd,nd)} of the same type and kind as {\ttfamily Solution}, representing the Cholesky upper-\/triangular decomposition of the matrix of coefficients $A$ of the linear system. This matrix is returned by the call to \mbox{\hyperlink{interfaceMatrixCholesky__mod_1_1getChoUpp}{get\+Cho\+Upp}}. \\
\hline
\mbox{\texttt{ out}}  & {\em Coef\+Mat\+Cho\+Dia} & \+: The input contiguous array of shape {\ttfamily (nd)} of the same type and kind as {\ttfamily Solution}, representing the diagonal elements of the Cholesky factorization of the matrix of coefficients $A$ of the linear system. This vector is returned by the call to \mbox{\hyperlink{interfaceMatrixCholesky__mod_1_1getChoUpp}{get\+Cho\+Upp}}. \\
\hline
\mbox{\texttt{ in}}  & {\em Right\+Hand} & \+: The input contiguous vector of size {\ttfamily nd} of the same type and kind as {\ttfamily Solution}, representing the right-\/hand-\/side $B$ in the system of linear equations to be solved ({\bfseries{optional}}, if missing, then $B$ must be provided as input in {\ttfamily Solution}).\\
\hline
\end{DoxyParams}

This condition is only checked for when the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1} is set at build time.

This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
