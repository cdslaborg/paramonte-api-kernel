\begin{DoxyWarning}{Warning}
The approximation is accurate with a relative error of 0.\+001 for any redshift above 0.\+1, or {\ttfamily zplus1 $>$= 1.\+1}. Note that for redshifts less than 0.\+1, the error in the calculated luminosity distance grows to more than 0.\+001. This algorithm should therefore not be used for zplus1$<$0.\+1.
\end{DoxyWarning}
\begin{DoxyRemark}{Remarks}
The distance is calculated according to approximate algorithm of Wickramasinghe \& Okwatta (2010).
