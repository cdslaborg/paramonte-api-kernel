Generate the Fisher Information of the Normal distribution.  




\doxysubsection{Detailed Description}
Generate the Fisher Information of the Normal distribution. 

The Fisher information for the Normal distribution is defined by the following equation, \begin{equation} \large \mathcal{I}(\mu,\sigma) = \begin{pmatrix} \frac{1}{\sigma^2} & 0 \\ 0 & \frac{2}{\sigma^2} \\ \end{pmatrix} \end{equation}


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em var\+Inv} & \+: The input scalar of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} representing the inverse of the variance of the Normal distribution. The positivity of the input value is checked only if the library is built with the preprocessor macro {\ttfamily CHECK\+\_\+\+ENABLED=1}.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily Fisher\+Info} \+: The output scalar of the same type and kind as the input {\ttfamily var\+Inv} representing the Fisher Information matrix of the Normal distribution.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
