{\footnotesize\ttfamily real(RK) function, dimension(nd,nd) Statistics\+\_\+mod\+::gen\+Rand\+Cor\+Mat (\begin{DoxyParamCaption}\item[{integer(IK), intent(in)}]{nd,  }\item[{real(RK), intent(in)}]{eta }\end{DoxyParamCaption})}



Return a random correlation matrix, sample correlation matrices {\ttfamily Rand\+Cor\+Mat} from a distribution proportional to {\ttfamily (det(\+C))$^\wedge$(eta−1)}. The larger the {\ttfamily eta}, the larger will be the determinant, meaning that generated correlation matrices will more and more approach the identity matrix. The value {\ttfamily eta = 1} corresponds to the uniform distribution. This procedure is based on the Vine approach described by Lewandowski, Kurowicka, and Joe (2009), \char`\"{}\+Generating random correlation matrices based on vines and extended onion method\char`\"{}. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em nd} & \+: The rank of the correlation matrix. \\
\hline
\mbox{\texttt{ in}}  & {\em eta} & \+: The parameter that roughly represents the shape parameters of the Beta distribution. The larger the value of {\ttfamily eta} is, the more homogeneous the correlation matrix will look. In other words, set this parameter to some small number to generate strong random correlations in the output random correlation matrix.\\
\hline
\end{DoxyParams}
\begin{DoxyReturn}{Returns}
{\ttfamily Rand\+Cor\+Mat} \+: A random correlation matrix.
\end{DoxyReturn}
\begin{DoxyWarning}{Warning}
The conditions {\ttfamily nd \texorpdfstring{$>$}{>} 1} and {\ttfamily eta \texorpdfstring{$>$}{>} 0.\+0} must hold, otherwise the first element of output, {\ttfamily gen\+Rand\+Cor\+Mat(1,1)}, will be set to {\ttfamily -\/1} to indicate the occurrence of an error.

If the Cholesky factorization fails, the component {\ttfamily Rand\+Cor\+Mat(1,1) = -\/1} of the output will be set to indicate the occurrence of an error.
\end{DoxyWarning}
