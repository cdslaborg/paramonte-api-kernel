Compute and return the Empirical Cumulative Distribution Function (ECDF) of the input univariate sample.  




\doxysubsection{Detailed Description}
Compute and return the Empirical Cumulative Distribution Function (ECDF) of the input univariate sample. 


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Sample} & \+: The contiguous array of shape {\ttfamily (np)} (rank {\ttfamily 1}) of either ~\newline
 ~ type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}} or ~\newline
 ~ type {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ae3573d46486b5f1a0a8d5033a9d12879}{IK8}}, \mbox{\hyperlink{namespaceConstants__mod_acd3c33e94d12f6e7d7c0ee79b2d6280b}{IK16}}, \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, or \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}} ~\newline
 ~ containing the univariate sample, where {\ttfamily np} is the number of data points in the sample. ~\newline
 ~ On return, the input {\ttfamily Sample} will be sorted in ascending order. \\
\hline
\mbox{\texttt{ out}}  & {\em ECDF} & \+: The output contiguous array of rank {\ttfamily 1} of the same size as the input {\ttfamily Sample} containing the Empirical Cumulative Distribution Function (ECDF) of the input univariate {\ttfamily Sample}. If the input {\ttfamily Sample} is of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}, then {\ttfamily ECDF} must be the same type and kind as {\ttfamily Sample}. If the input {\ttfamily Sample} is of type {\ttfamily integer} of kind \mbox{\hyperlink{namespaceConstants__mod_ae3573d46486b5f1a0a8d5033a9d12879}{IK8}}, \mbox{\hyperlink{namespaceConstants__mod_acd3c33e94d12f6e7d7c0ee79b2d6280b}{IK16}}, \mbox{\hyperlink{namespaceConstants__mod_ab12a86d74008f0d7f287c3564c3defd9}{IK32}}, or \mbox{\hyperlink{namespaceConstants__mod_ad6aaf2f477491dc84e7ebbab3b64694e}{IK64}}, then {\ttfamily ECDF} must be of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}. \\
\hline
\mbox{\texttt{ out}}  & {\em Lower} & \+: The output contiguous array of the same type, kind, rank, and size as the output {\ttfamily ECDF} containing the lower confidence bound on the ECDF as the specified $1-\alpha$ probability ({\bfseries{optional}}, if missing, it is not computed). \\
\hline
\mbox{\texttt{ out}}  & {\em Upper} & \+: The output contiguous array of the same type, kind, rank, and size as the output {\ttfamily ECDF} containing the upper confidence bound on the ECDF as the specified $1-\alpha$ probability ({\bfseries{optional}}, if missing, it is not computed). \\
\hline
\mbox{\texttt{ in}}  & {\em alpha} & \+: The input scalar of type {\ttfamily real} of the same kind as {\ttfamily ECDF} from which the probability $1-\alpha$ for the upper and lower confidence bounds of ECDF is computed. For example, if $\alpha = 0.05$, then the output {\ttfamily Lower} and {\ttfamily Upper} confidence bounds contain the CDF of the true parent distribution of the sample with $ 95\% $ probability. In such a case, the lower bound marks the $ \alpha / 2 $ probability contour and the upper bound marks the $ 1 - \alpha / 2 $ probability contour ({\bfseries{optional}}, default = {\ttfamily 0.\+05}). \\
\hline
\mbox{\texttt{ in}}  & {\em sorted} & \+: The input {\ttfamily logical} of default kind. If {\ttfamily .true.}, the sample is assumed to be sorted on input and will not be sorted. If the input {\ttfamily Sample} is already sorted, set {\ttfamily sorted = .true.} as it will lead to much faster ECDF computations ({\bfseries{optional}}, default = {\ttfamily .false.}, that is, the sample will be sorted by the procedure prior to computing the ECDF).\\
\hline
\end{DoxyParams}
\begin{DoxyWarning}{Warning}
This procedure is {\ttfamily impure} when the library is compiled with macro {\ttfamily CHECK\+\_\+\+ENABLED=1}. By default, it is {\ttfamily pure} in {\ttfamily release} build and {\ttfamily impure} in {\ttfamily debug} and {\ttfamily testing} builds.
\end{DoxyWarning}
