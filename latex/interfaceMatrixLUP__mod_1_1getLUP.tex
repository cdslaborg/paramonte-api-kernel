\doxysubsection{Detailed Description}
Return the LU-\/\+Pivoted decomposition of the input square matrix {\ttfamily Matrix\+LUP(nd,nd)}. The factorization has the form. 

\begin{equation} \text{MatrixLUP} = P * L * U \end{equation} where $P$ is the permutation matrix, $L$ is lower triangular with unit diagonal elements (that are missing in the output {\ttfamily Matrix\+LUP}), and $U$ is upper triangular of the decomposition. {\bfseries{LUP}} stands for the LU factorization with partial Pivoting.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in,out}}  & {\em Matrix\+LUP} & \+: The input contiguous square matrix of shape {\ttfamily (nd,nd)} of type {\ttfamily real} of kind \mbox{\hyperlink{namespaceConstants__mod_aaf5c6c840ed0befe5a37da626c35716f}{RK32}}, \mbox{\hyperlink{namespaceConstants__mod_a855643f887b4c787e6542e9e7849680f}{RK64}}, or \mbox{\hyperlink{namespaceConstants__mod_aeef7c44b90b75aeeebdef1da504886cb}{RK128}}. ~ On output, it is completely overwritten by its Pivoted LU decomposition. \\
\hline
\mbox{\texttt{ out}}  & {\em Row\+Perm} & \+: The output contiguous vector of size {\ttfamily nd} of type {\ttfamily integer} of default kind \mbox{\hyperlink{namespaceConstants__mod_a2bf4d6767c9954d3a495d10ab6b538d6}{IK}} containing the pivot indices. ~\newline
 ~ For {\ttfamily 1 \texorpdfstring{$<$}{<}= i \texorpdfstring{$<$}{<}= nd}, the {\ttfamily i}th row of the matrix is interchanged with row {\ttfamily Row\+Perm(i)}. \\
\hline
\mbox{\texttt{ out}}  & {\em parity} & \+: An output real of the same type and kind as the input {\ttfamily Matrix\+LUP} whose value is {\ttfamily +1} if the number of row interchanges is even, otherwise it is {\ttfamily -\/1}. \\
\hline
\mbox{\texttt{ out}}  & {\em failed} & \+: The output {\ttfamily logical} of default kind that is {\ttfamily .true.} if a singular matrix is detected, indicating the LUP decomposition failure.\\
\hline
\end{DoxyParams}
\begin{DoxyRemark}{Remarks}
This routine can be used in combination with \mbox{\hyperlink{interfaceLinSys__mod_1_1solveLinSysLUP}{solve\+Lin\+Sys\+LUP}} to solve linear equations or invert a matrix.
\end{DoxyRemark}
