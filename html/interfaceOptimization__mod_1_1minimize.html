The constructor of the class BrentMinimum_type. Compute the minimum of the input 1-dimensional function isolated to a fractional precision of about xtol using Brent's method.

Parameters
[in]getFunc: The 1-dimensional function which will have to be minimized.
[in]x0: The lower point in the set of optional bracketing triplet of abscissas that bracket the minimum of the function such that, x0 < x1 < x2 and getFunc(x0) > getFunc(x1) < getFunc(x2) (optional).
[in]x1: The middle point in the set of optional bracketing triplet of abscissas that bracket the minimum of the function such that, x0 < x1 < x2 and getFunc(x0) > getFunc(x1) < getFunc(x2) (optional).
[in]x2: The upper point in the set of optional bracketing triplet of abscissas that bracket the minimum of the function such that, x0 < x1 < x2 and getFunc(x0) > getFunc(x1) < getFunc(x2) (optional).
[in]xtol: An optional fractional precision within which is the minimum is returned. The default value is sqrt(epsilon(1._RK)).
Returns
BrentMinimum : An object of class BrentMinimum_type that contains the minimum of the function (xmin) and the function value at the minimum (fmin) as well as other relevant information.
