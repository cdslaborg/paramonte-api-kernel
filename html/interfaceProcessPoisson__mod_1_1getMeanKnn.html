Return the average nearest-neighbor distance in dimension nd with the rate density logDensity.

Parameters
[in]nd: The number of dimensions.
[in]ndInverse: The inverse of nd (i.e., 1._RK / nd).
[in]logDensity: The natural logarithm of the target density.
[in]logVolUnitBall: The log-volume of unit nd-ball.
Returns
logExpectedMinPairDist : A symmetric matrix of shape (np,np) containing the pairwise distances-squared of the points from each other.
Remarks
The input logVolUnitBall can be computed via GeometryEllipsoid_mod::genLogVolUnitBall() and passed to this routine.
