Detailed Description

Generate and return a vector of shape (nd) or a matrix of shape (nd,np) of randomly uniformly distributed points from within an ellipsoid specified by the Lower triangle and diagonals of the Cholesky factorization of its representative matrix. Here, nd represents the number of dimensions of the ellipsoid and np represents the number of random vectors to generate. The elements of each uniformly distributed output random vector Rand(i), i=1:nd are guaranteed to be in the range,.

Mean(i) - sqrt(RepMat(i,i)) < Rand(i) < Mean(i) + sqrt(RepMat(i,i))

or in case of a 2D output matrix,

Mean(i) - sqrt(RepMat(i,i)) < Rand(i,j) < Mean(i) + sqrt(RepMat(i,i))

where RepMat is the representative matrix of the ellipsoid whose Cholesky lower triangle and diagonal elements are represented by ChoLow and ChoDia.

Parameters
[out]Rand: The output randomly uniformly distributed contiguous array of shape (nd) or (nd,np) of type real of kind RK32, RK64, or RK128, where nd represents the number of dimensions of the ellipsoid and np number of random vectors to generate.
[in]Mean: The input array of shape (nd) of type real of kind RK32, RK64, or RK128 representing the center of the ellipsoid (optional, default = [( 0._RK, i = 1, nd )]).
[in]ChoLow: The input square matrix of shape (nd,nd) of type real of kind RK32, RK64, or RK128 containing lower triangle of the Cholesky factorization of the representative matrix of the ellipsoid. Note that only the lower triangle of ChoLow is used and the upper triangle and diagonals are ignored. (optional, default = the identity matrix. If missing, ChoDia must be also missing.)
[in]ChoDia: The input array of shape (nd) of type real of kind RK32, RK64, or RK128 containing the diagonal elements of the Cholesky factorization of the representative matrix of the ellipsoid. (optional, default = the identity matrix. If missing, ChoLow must be also missing.)
[in]logVolUnitBall: The natural logarithm of the volume of an nd-dimensional ball of unit radius. (optional. If present, random vectors will be drawn from an ellipsoid of unit-volume.)

program test
implicit none
use Constants_mod, only: IK, RK ! (Here `RK` can point to any of \RK32, \RK64, \RK128)
use MatrixDiag_mod, only: genDiag
use MatrixCholesky_mod, only: getChoLow
use GeometryEllipsoid_mod, only: getLogVolUnitBall
use DomainEllipsoid_mod, only: getRandUnif
integer(IK) , parameter :: NDIM = 3_IK ! the number of dimensions of the ellipsoid.
integer(IK) , parameter :: NSIM = 1000 ! the number of dimensions of random vectors to generate.
real(RK) :: Rand(NDIM), RandSample(NDIM,NSIM), logVolUnitBall
real(RK) , allocatable :: Mean(:), ChoLowCovUpp(:,:), ChoDia(:)
call getRandUnif(Rand) ! generate a single NDIM-dimensional uniform random vector from a sphere of unit-radius.
call getRandUnif(RandSample) ! generate NSIM NDIM-dimensional uniform random vectors from a sphere of unit-radius.
call getLogVolUnitBall(NDIM,logVolUnitBall)
call getRandUnif(Rand, logVolUnitBall = logVolUnitBall) ! generate a single NDIM-dimensional uniform random vector from a sphere of unit-volume.
call getRandUnif(RandSample, logVolUnitBall = logVolUnitBall) ! generate NSIM NDIM-dimensional uniform random vectors from a sphere of unit-volume.
Mean = [( real(i,RK), integer :: i = 1,NDIM )]
call getRandUnif(Rand, Mean) ! generate a single NDIM-dimensional uniform random vector with the input Mean Vector.
call getRandUnif(RandSample, Mean) ! generate NSIM NDIM-dimensional uniform random vectors with the input Mean Vector.
ChoLowCovUpp = genDiag(NDIM, NDIM, diag = Mean, offDiag = 0.6) ! The representative matrix of the ellipsoid.
call getChoLow(NDIM,ChoLowCovUpp,ChoDia) ! Compute the Cholesky factorization of the representative matrix.
call getRandUnif(Rand, Mean, ChoLowCovUpp) ! generate a single NDIM-dimensional uniform random vector with the input Mean Vector and shape structure.
call getRandUnif(RandSample, Mean, ChoLowCovUpp, ChoDia) ! generate NSIM NDIM-dimensional uniform random vectors with the input Mean Vector and shape structure.
call getRandUnif(Rand, Mean, ChoLowCovUpp, logVolUnitBall) ! generate a single NDIM-dimensional uniform random vector with the input Mean Vector and shape structure, but with a unit volume.
call getRandUnif(RandSample, Mean, ChoLowCovUpp, ChoDia, logVolUnitBall) ! generate NSIM NDIM-dimensional uniform random vectors with the input Mean Vector and shape structure, but with a unit volume.
end program test
