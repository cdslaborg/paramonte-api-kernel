Return the normalized data with respect to the input mean vector of length nd.

Parameters
[in]nd: The number of dimensions of the input sample.
[in]np: The number of points in the sample.
[in]Mean: The mean vector of length nd.
[in]Point: The array of shape (nd,np) containing the sample.
Returns
NormData : The output normalized points array of shape (np,nd).
Remarks
Note the difference in the shape of the input Point vs. the output NormData.

Return the normalized 2D data of size (nd,np) with respect to the mean of the data along the second dimension of length np.

Parameters
[in]nd: The length of the input Data matrix along the first dimension.
[in]np: The length of the input Data matrix along the second dimension.
[in]Data: The input data series data vector.
[in]Weight: The vector of weights of the input data points (optional, default = array of ones).
[in]tenabled: A logical value that, if .true. will cause the output NormData to have transposed shape of the input Point(nd,np) matrix, that is (np,nd) (optional, default = .false.).
Returns
NormData : The integrated autocorrelation (IAC) via the BatchMeans method.
Remarks
Note that np must be large enough to get a meaningful answer.
