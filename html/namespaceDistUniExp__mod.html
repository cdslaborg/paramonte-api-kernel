Detailed Description

This module contains the classes and procedures for setting up and computing a grid-like objective function that is formed by multiple (potentially elongated) Multivariate Normal (MVN) Distributions.

Author
Amir Shahmoradi

PURE type(DistUniExp_type) function DistUniExp_mod::construct ( real(RK), intent(in), optional  lambda)

Set the DistUniExp properties and return an object of type DistUniExp_type containing the objective function with the requested properties.

Parameters
[in]lambda: The rate parameter of the exponential distribution (optional, default = 1.). WARNING: The rate parameter must be a positive number. This condition will be checked only if the library is built with macro CHECK_ENABLED.
Returns
self : An object of type DistUniExp_type containing the objective function with the requested properties.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
