Detailed Description

Generate and return a vector of shape (nd) or a matrix of shape (nd,np) of randomly uniformly distributed points from within an ellipsoid specified by the Lower triangle and diagonals of the Cholesky factorization of its representative matrix. Here, nd represents the number of dimensions of the ellipsoid and np represents the number of random vectors to generate. The elements of each uniformly distributed output random vector Rand(i), i=1:nd are guaranteed to be in the range,.

Mean(i) - sqrt(RepMat(i,i)) < Rand(i) < Mean(i) + sqrt(RepMat(i,i))

or in case of a 2D output matrix,

Mean(i) - sqrt(RepMat(i,i)) < Rand(i,j) < Mean(i) + sqrt(RepMat(i,i))
Parameters
[in]nd: The number of dimensions of the ellipsoid.
[in]np: The number of random vectors to generate (optional, default = 1).
[in]Mean: The input array of shape (nd) of type real of kind RK32, RK64, or RK128 representing the center of the ellipsoid (optional, default = [( 0._RK, i = 1, nd )]).
[in]ChoLow: The input square matrix of shape (nd,nd) of type real of kind RK32, RK64, or RK128 containing lower triangle of the Cholesky factorization of the representative matrix of the ellipsoid. Note that only the lower triangle of ChoLow is used and the upper triangle and diagonals are ignored. (optional, default = the identity matrix. If missing, ChoDia must be also missing.)
[in]ChoDia: The input array of shape (nd) of type real of kind RK32, RK64, or RK128 containing the diagonal elements of the Cholesky factorization of the representative matrix of the ellipsoid. (optional, default = the identity matrix. If missing, ChoLow must be also missing.)
[in]logVolUnitBall: The natural logarithm of the volume of an nd-dimensional ball of unit radius. (optional. If present, random vectors will be drawn from an ellipsoid of unit-volume.)
Returns
Rand : The randomly uniformly distributed array of shape (nd) or (nd,np) of type real of kind RK32, RK64, or RK128.
