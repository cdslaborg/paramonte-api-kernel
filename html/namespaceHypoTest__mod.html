real(RK)  Point,

real(RK)  probKS

Return the one-sample Kolmogorov–Smirnov (KS) test results for the null hypothesis that the data in vector Point comes from a distribution whose CDF is specified by the input genCDF() function.

Parameters
[in]np: The number of points in the input vector Point.
[in,out]Point: The sample. On return, this array will be sorted in Ascending order.
[in]genCDF: The function returning the Cumulative Distribution Function (CDF) of the sample.
[out]statKS: The KS test statistic.
[out]probKS: The p-value of the test, returned as a scalar value in the range [0,1]. The output probKS is the probability of observing a test statistic as extreme as, or more extreme than, the observed value under the null hypothesis. Small values of probKS cast doubt on the validity of the null hypothesis.

◆ getSampleDisparity()

pure subroutine HypoTest_mod::getSampleDisparity ( integer(IK)  Index,
real(RK)  SortedSample,
real(RK)  ProbKS,
real(RK)  StatKS,
real(RK)  Lambda,
type(Err_type)  Err 
)

Return the weights of each of the points in the input vector of Point((2:np) as ProbKS. These KS two-sample test probabilities are obtained by dividing the input sample into approximately equal sizes and computing the quantities of interest.

Parameters
[in]Index: The index of the points in the input Point for which the weights will be computed. The smaller the length of this vector, the less it will take to return the results. In return, the corresponding probabilities and statistics for the missing points can be approximated via linear interpolation from the existing neighbors.
[in]SortedSample: The input array of points of shape (np) sorted in order of relevance (unbiasedness).
[out]ProbKS: The output vector of the same length as Index containing the two-sample KS test probabilities (the wieghts).
[out]StatKS: The output vector of the same length as Index containing the two-sample KS test statistics.
[out]lambda: The output vector of the same length as Index containing the sample-size-adjusted two-sample KS test statistics.
[out]Err: An object of class Err_type containing error information.
Remarks
The output vector of probabilities and the corresponding KS statistics are not unreliable for any value of Index that is less than 8, since the corresponding effective sample sizes N_e are less than four.
For any value Index that is unity, the effective sample size will be 0: \( N_e = \frac{N_1 \times N_2}{N_1 + N_2} \). As such, for the sake of comprehensiveness, its KS probability is assigned a value of 1 and KS statistic of 0.
