type(Timer_type) function Timer_mod::Timer_type::constructTimer ( logical, intent(out), optional  failed)

This is the constructor of the class Timer_type. Before returning the object, this function also calls the tic() method of the Timer_type object to reset the timer.

Parameters
[out]failed: An output logical of default kind (optional. If missing, the program will halt by calling error stop if there is no system clock). An output value of .true. implies that the system does not possess a processor clock.
Returns
Timer : An object of class Timer_type.
