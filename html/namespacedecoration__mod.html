Wrap the input text with ANSI/VT100 Control sequences to stylize and color the string.

Parameters
[in]string: The input string to be stylized.
[in]attr: The requested attribute (optional). It can be:
  • "off" : All attributes off (0).
  • "bold" : Boldface text (1)
  • "bright" : Bright text (1)
  • "dim" : Dimmed text (2)
  • "underlined" : Underlined text (4)
  • "blinking" : Blinking text (5)
  • "reverse" : Reversed attributes text (7)
  • "hidden" : Hidden text (8) The default value is "off".
[in]clfg: The Foreground color of the text (optional, see below for possible colors).
[in]clbg: The Background color of the text (optional, see below for possible colors).
Returns
modifiedString : The output string wrapped with the requested style and coloring.
Remarks
Possible Foreground / Background colors are the following:
  • "black" : (30) / (40)
  • "red" : (31) / (41)
  • "green" : (32) / (42)
  • "yellow" : (33) / (43)
  • "blue" : (34) / (44)
  • "magenta" : (35) / (45)
  • "cyan" : (36) / (46)
  • "light gray" : (37) / (47)
  • "dark gray" : (90) / (100)
  • "light red" : (91) / (101)
  • "light green" : (92) / (102)
  • "light yellow" : (93) / (103)
  • "light blue" : (94) / (104)
  • "light magenta" : (95) / (105)
  • "light cyan" : (96) / (106)
  • "white" : (97) / (107)
