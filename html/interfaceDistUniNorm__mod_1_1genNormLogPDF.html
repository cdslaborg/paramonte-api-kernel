Generate the natural logarithm of probability density function (PDF) of the Normal distribution. More...

Detailed Description

Generate the natural logarithm of probability density function (PDF) of the Normal distribution.

Parameters
[in]Point: The input scalar or contiguous array of rank 1 of type real of kind RK32, RK64, or RK128, representing the point(s) at which the PDF must be computed.
[in]mean: The input scalar of the same type and kind as Point representing the mean of the distribution (optional, default = 0._RK).
[in]invStd: The input scalar of the same type and kind as Point representing the inverse of the standard deviation of the distribution The positivity of the input value is checked only if the library is built with the preprocessor macro CHECK_ENABLED=1. (optional, default = 1._RK. If present, then the input argument mean must be also present).
[in]logInvStd: The input scalar of the same type and kind as Point representing the natural logarithm of the inverse of the standard deviation of the distribution. (optional, default = 0._RK. If present, then the input arguments mean and invStd must be also present).
Returns
LogPDF : The output scalar of the same type, kind, and rank as the input Point representing the PDF of the Normal distribution at the given input Points.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
Note
Since the PDF values are often extremely small, the values are returned as natural logarithms of the actual PDF values. Converting the output to natural values is as simple as passing it to the Fortran intrinsic procedure exp().
The fact that the input argument mean is required to be present when stdInd is present has good generic programming and performance reasons (over the alternative of making it an optional input argument when invStd is present).
