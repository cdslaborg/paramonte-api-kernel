Perform the Kmeans clustering on the input data set represented by the array Point(nd, np) and return the resulting cluster characteristics as an object of type Kmeans_type.

Parameters
[in]nd: See the description of the runKmeans.
[in]np: See the description of the runKmeans.
[in]nc: See the description of the runKmeans.
[in]Point: See the description of the runKmeans.
[in]InitCenter: See the description of the runKmeans.
[in]niterMax: See the description of the runKmeans.
[in]nszciMax: See the description of the runKmeans.
[in]relTol: See the description of the runKmeans.
Returns
Kmeans : An object of type Kmeans_type containing the properties of the identified clusters. The Errstat component of the output Kmeans will be set to 1 if the number of iterations reaches the input niterMax or its default value before convergence occurs. The Errstat component of the output Kmeans will be set to 2 if more than nzsciMax clustering tries result in clusters with zero members. In both cases in the above, Erroccurred = .true. upon exiting the procedure.
Warning
The input value for niterMax must be larger than 0 since at least one iteration will be performed by the procedure.
This algorithm does not check for the consistency of the input values for nc, nd, and np. It is currently the user's responsibility to ensure that these input values are logical and sensible.

Perform the Kmeans clustering for nt tries on the input data set represented by the array Point(nd, np) and return the clustering that yields the least potential.

Parameters
[in]nd: See the description of the runKmeans.
[in]np: See the description of the runKmeans.
[in]nc: See the description of the runKmeans.
[in]nt: The number of independent instances of Kmeans to run.
[in]Point: See the description of the runKmeans.
[in]InitCenter: See the description of the runKmeans.
[in]niterMax: See the description of the runKmeans.
[in]nszciMax: See the description of the runKmeans.
[in]relTol: See the description of the runKmeans.
Returns
Kmeans : An object of type Kmeans_type containing the properties of the identified clusters. The Errstat component of the output Kmeans will be set to 1 if the number of iterations reaches the input niterMax or its default value before convergence occurs. The Errstat component of the output Kmeans will be set to 2 if more than nzsciMax clustering tries result in clusters with zero members. In both cases in the above, Erroccurred = .true. upon exiting the procedure.
Warning
The input value for nt must be at least 1, otherwise it will lead to a runtime error or at best, the output will be undefined.
The input value for niterMax must be larger than 0 since at least one iteration will be performed by the procedure.
This algorithm does not check for the consistency of the input values for nc, nd, and np. It is currently the user's responsibility to ensure that these input values are logical and sensible.
