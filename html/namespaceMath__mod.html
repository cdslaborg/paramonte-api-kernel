real(RK)  logValueSmaller 
)

Return log( exp(logValueLarger) - exp(logValueSmaller) ) robustly (without overflow or underflow).

Parameters
[in]logValueLarger: A real number.
[in]logValueSmaller: A real number.
Returns
logSubExp : A real number.
Warning
The onus is on the user to ensure logValueLarger > logValueSmaller.
Remarks
This function is very useful for situations where exp(logValueLarger) is likely to cause overflow.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
