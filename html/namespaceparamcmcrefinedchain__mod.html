Return the refined Markov chain, given the input Markov chain and its specifications. This procedure is a method of the RefinedChain_type class.

Parameters
[in,out]RefinedChain: An object of class RefinedChain_type.
[in,out]CFC: An object of type ChainFileContents_type containing the Markov chain.
[out]Err: An object of class Err_type indicating whether any error has occurred or not.
[in]burninLoc: The estimated location of burnin point in the Markov chain (optional). If not provided, it will be extracted from the components of the input CFC.
[in]refinedChainSize: The requested refined sample size (optional). If the size of the refined sample is given as input, then the requested sample is directly generated based on the input size.
[in]sampleRefinementCount: The maximum number of times the sample can be refined (optional, default = Infinity). : For example, if set to 1, then only one round of refinement will be performed on the Markov chain.
[in]sampleRefinementMethod: The requested method of refining the sample (optional, default = "BatchMeans").

Return the refined vector of weights of the vector of weights of a weighted Markov chain.

Parameters
[in]np: The number of elements of the Weight vector.
[in]Weight: The input vector of weights.
[in]skip: The size of the jumps that have to be made through the weighted Markov chain.
[in]refinedChainSize: The requested refined sample size (optional). If present, then the refined chain (represented by the : vector Weight) will be refined such that the resulting refined chain has the size refinedChainSize.
Returns
RefinedWeight : An array of size np, whose elements indicate which points are present in the final refined chain.
Examples:
skip: 1
weight: 5, 0, 1, 3, 1
refinedweight: 5, 0, 1, 3, 1
skip: 2
weight: 5, 0, 1, 3, 1
refinedweight: 3, 0, 0, 2, 0
skip: 3
weight: 5, 0, 1, 3, 1
refinedweight: 2, 0, 0, 1, 1

Return the best skip size through a Markov chain to refined it to the optimal requested size.

Parameters

Returns
skip4NewSampleSize : The computed skip size.
Warning

Write the computed refined chain to the specified output file.

Parameters
[in]sampleFilePath: The path to the input chain file that must be read.
[in]delimiter: The delimiter used in the file.
[in]ndim: The number of dimensions of the sampled states in the sample file. This is basically, the size of the domain of the objective function.
[in]tenabled: An optional input logical value standing for transpose-enabled. If .false., the input data will be naturally read according to Fortran column-wise data storage rule as a matrix of rank 0:nd * 1:np. If .false., the input sample file will be read as a matrix of rank 1:np * 0:nd. Note that np represents the number of rows in the files (that is, the number of sampled points, whereas nd represents the number of columns in the input file (optional, default = .false.).
Returns
RefinedChain : An object of class RefinedChain_type containing the sampled states read from the specified input file.
