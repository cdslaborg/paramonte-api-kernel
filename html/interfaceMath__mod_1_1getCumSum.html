Detailed Description

Return the cumulative sum of the input Array, optionally in the backward direction and optionally, reverse the output cumulative sum upon return.

Parameters
[in]Array: The input contiguous array of shape (:) of either type real of kind RK32, RK64, or RK128 or type integer of kind IK8, IK16, IK32, or IK64 whose cumulative sum will have to be computed.
[out]CumSum: The output contiguous array of the same size, shape, type, and kind as the input Array containing the cumulative sum of Array in the specified direction.
[in]backward: The input scalar of type logical of default kind. If .true., the output CumSum will be computed from the last element to the first element of the input Array (optional, default = .false.).
[in]reversed: The input scalar of type logical of default kind. If .true., the output array CumSum will be reversed upon return, such that its last element becomes the first (optional, default = .false.. If present, then the optional input argument backward must be also present).
Warning
The size of the Array and Cumsum arguments must be the same. This condition is only checked when the preprocessor macro CHECK_ENABLED=1 is set.
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
