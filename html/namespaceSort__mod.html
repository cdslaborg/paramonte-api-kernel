Detailed Description

This module contains procedures for sorting arrays and finding quantiles of arrays. Recommended routines for sorting arrays: sortAscending_RK and sortQNR are among the fastest sorting algorithms. However, sortQNR has the benefit of only taking one argument and not returning any error flags to inspect. For a performance comparison of the algorithms see: Test_Sort_mod::test_performance_1.
