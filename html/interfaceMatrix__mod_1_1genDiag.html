Generate a diagonal matrix of shape (n,m) with the input diagonal and off-diagonal values.

Parameters
[in]n: The number of rows of the output diagonal matrix.
[in]m: The number of columns of the output diagonal matrix.
[in]Diag: The input vector of length min(n,m) whose elements will appear on the diagonal elements of the output matrix.
[in]offdiag: The value to appear on the off-diagonal elements of the output eye matrix (optional, default = 0.).
Returns
DiagMat : A matrix of n rows and m columns whose diagonal elements are set to the input Diag vector and whose off-diagonal elements are set to the optional input offdiag value, if provided as input.

Parameters

[in]m: The number of columns of the eye matrix.
[in]diag: The value to appear on the diagonal elements of the output eye matrix (optional, default = 1.).
[in]offdiag: The value to appear on the off-diagonal elements of the output eye matrix (optional, default = 0.).
Returns
eye : A matrix of n rows and m columns whose diagonal elements are set to the input diag value or, if not provided, to 1..
