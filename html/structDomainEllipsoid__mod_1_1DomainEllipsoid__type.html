Construct an object of type DomainEllipsoid_type.

Parameters
[in]ndim: The number of dimensions of the domain.
[in]Center: The center of the ellipsoidal domain (optional, default = A vector of all zeros).
[in]RepUpp: The upper triangle and diagonal elements of the representative matrix of the ellipsoidal domain. The lower triangle is not used (optional, default = Identity matrix).
[in]Err: An object of type Err_mod::Err_type containing error information. (optional, if not provided and an error occurs, it will lead to a complete program halt).
Returns
self : An object of type DomainEllipsoid_type.
Remarks
If not readily available, the input representative matrix RepUpp can be constructed by calling Sample_mod::genCovUppFromCorUpp().
This routine computes the inverse of the input representative matrix. As such, if an error occurs, the first element of selfChoLow (selfChoLow(1,1)) will be set to a negative number. Additionally, if the input argument Err is present, an error message will be returned in Errmsg and Erroccurred = .true..
When the domain is non-spherical, radiusSq is set to negative infinity.
