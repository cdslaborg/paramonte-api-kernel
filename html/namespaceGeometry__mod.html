Return the estimated mean and standard deviation of the volume occupied by a set of np points uniformly distributed in an nd dimensional domain of arbitrary shape and size.

Parameters
[in]nd: The number of dimensions.
[in]np: The number of input points.
[in]Point: The input array of points of shape (nd,np).
[in]RefPoint: The input array of length (nd) representing the point from which distances differences will be computed.
[out]DistSortedExpDiff: The array of length np of differences of adjacent distances-from-RefPoint-exponentiated-to-nd.
[out]Index: The output array of length (np) such that Point(:,Index) is ordered similar to DistSortedExpDiff.
[out]failed: The output logical flag indicating whether sorting of the distances and therefore, the procedure has failed. If failed = .true., the procedure has returned without computing the output DistSortedExpDiff vector.
Remarks
The term Exp in the name of the procedure stands for * Distance-Sorted Exponentiated to nd* before computing the differences.

Compute the effective total volume of all bounded ellipsoidal regions together while considering the possible overlaps. This computation is done aggressively via Monte Carlo simulation.

Parameters
[in]nd: The number of dimensions.
[in]nc: The number of input clusters.
[in]relTol: The accuracy tolerance of the computed effective sum of volumes (NOT the log-transformed of the sum).
[in]LogVol: An array of size (nc) representing the logarithms of the volumes of the ellipsoids normalized to the volume of an nd-ball.
[in]Center: An array of size (nd,nc) representing the centers of the ellipsoids.
[in]ChoDia: An array of size (nd,nc) representing the diagonal elements of the Cholesky lower triangle of the covariance matrices of the ellipsoids.
[in]logSumVol: = sum(LogVol).
[in]InvCovMat: An array of size (nd,nd,nc) representing the inverse covariance matrices of the ellipsoids.
[in]ChoLowCovUpp: An array of size (nd,nd,nc) whose lower-triangle represents the Cholesky lower triangle of the covariance matrices of the ellipsoids.
[in,out]logSumVolEff: The logarithm of the sum of LogVolNormed. On output, it will be rewritten with the effective log-sum of the (normalized) volumes.
[in]ScaleFactorSq: An array of size (nc) representing the maximum of Mahalanobis-distance-squared of the members of each cluster from its center. This is needed if the ellipsoid properties have not been scaled, i.e., ScaleFactorSq > 1 (optional, the default is ScaleFactorSq(1:nc) = 1).
