Construct an object of type DomainEllipsoid_type.

Parameters
[in]ndim: The number of dimensions of the domain.
[in]Center: The center of the ellipsoidal domain (optional, default = A vector of all zeros).
[in]CovUpp: The upper triangle and diagonal elements of the covariance matrix of the ellipsoidal domain. The lower triangle is not used (optional, default = Identity matrix).
[in]Err: An object of type Err_mod::Err_type containing error information. (optional, if not provided and an error occurs, it will lead to a complete program halt).
Returns
self : An object of type DomainEllipsoid_type.
Remarks
If not readily available, the input covariance matrix CovUpp can be constructed by calling Sample_mod::getCovUppFromCorUpp().
This routine computes the inverse of the input covariance matrix. As such, if an error occurs, the first element of selfChoLow (selfChoLow(1,1)) will be set to a negative number. Additionally, if the input argument Err is present, an error message will be returned in Errmsg and Erroccurred = .true..
When the domain is non-spherical, radiusSq is set to negative infinity.

Given the Cholesky Lower triangle and diagonals of a given covariance matrix, this function return one point uniformly randomly drawn from inside of an nd-ellipsoid, whose nd elements RandMVU(i), i=1:nd are guaranteed to be in the range:

Mean(i) - sqrt(CovMat(i,i)) < RandMVU(i) < Mean(i) + sqrt(CovMat(i,i))
Parameters
[in]nd: The number of dimensions of the MVU domain.
[in]Mean: The mean vector of the MVU domain.
[in]ChoLow: The Cholesky lower triangle of the covariance matrix of the MVU domain.
[in]ChoDia: The diagonal elements of the Cholesky lower triangle of the covariance matrix of the MVU domain.
Returns
RandMVU : The randomly generated MVU vector.

Given the Cholesky Lower triangle and diagonals of a given covariance matrix, this function return one point uniformly randomly drawn from inside of an nd-ellipsoid, whose nd elements RandMVU(i), i=1:nd are guaranteed to be in the range:

Mean(i) - sqrt(CovMat(i,i)) < RandMVU(i) < Mean(i) + sqrt(CovMat(i,i))
Parameters
[in]nd: The number of dimensions of the MVU domain.
[in]Mean: The mean vector of the MVU domain.
[in]ChoLow: The Cholesky lower triangle of the covariance matrix of the MVU domain.
[in]ChoDia: The diagonal elements of the Cholesky lower triangle of the covariance matrix of the MVU domain.
[in]np: The number of random vectors to generate.
Returns
RandMVU : The matrix of rank (nd,np) each column of which will be one randomly generated MVU vector.
