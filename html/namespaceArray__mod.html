Flatten the input Point array such that each element of the output FlattenedPoint array has the same unity weight as elements in the array.

Parameters
[in]Point: The array of shape (np) containing the sample.
[in]Weight: The vector of length np containing the weights of points in the sample. The values of elements of Weight are allowed to be negative, in which case, the corresponding elements will be excluded from the output FlattenedPoint.
[in]sumWeight: The sum of all positive elements of Weight(np) (optional, will be computed if not provided).
Warning
Note the shape of the input argument Point(np).
Returns
FlattenedPoint : The flattened array of length sumWeight whose elements all have the same weight.

Returns count evenly spaced points over the interval [x1, x2] if x1 < x2, or [x2, x1] if x2 < x1.

Parameters
[in]x1: The starting value of the sequence.
[in]x2: The ending value of the sequence.
[in]count: The length of LinSpace to generate.
[in]fopen: A logical value. If .true., the first point will NOT be in the output LinSpace sequence. (optional, default = .false.).
[in]lopen: A logical value. If .true., the Last point will NOT be in the output LinSpace sequence. (optional, default = .false.).
Returns
LinSpace : An evenly-spaced array of length count within the interval specified by x1 and x2.


Example usage

use Array_mod, only: genLinSpace
write(*,*) genLinSpace(0._RK, 10._RK, 4, .false., .false.) ! = [0.00000000, 3.33333325, 6.66666651, 10.0000000] (default behavior)
write(*,*) genLinSpace(0._RK, 10._RK, 4, .false., .true.) ! = [0.00000000, 2.50000000, 5.00000000, 7.50000000]
write(*,*) genLinSpace(0._RK, 10._RK, 4, .true., .false.) ! = [2.50000000, 5.00000000, 7.50000000, 10.0000000]
write(*,*) genLinSpace(0._RK, 10._RK, 4, .true., .true.) ! = [1.25000000, 3.75000000, 6.25000000, 8.75000000]

Returns count evenly-logarithmically-spaced points over the interval [base**logx1, base**logx2] if logx1 < logx2, or over the interval [logx2, logx1] if logx2 < logx1.

Parameters
[in]logx1: The starting exponent such that base**logx1 is the starting value of the sequence.
[in]logx2: The ending exponent such that base**logx1 is the last value of the sequence.
[in]count: The length of LogSpace to generate.
[in]fopen: A logical value. If .true., the first point will NOT be in the output LogSpace sequence. (optional, default = .false.).
[in]lopen: A logical value. If .true., the last point will NOT be in the output LogSpace sequence. (optional, default = .false.).
[in]base: The base of the logarithm (oprional, default = exp(1), Euler`s number).
Returns
LogSpace : An evenly-spaced array of length count within the interval specified by logx1 and logx2.


Example usage

use Array_mod, only: genLogSpace
write(*,*) log( genLogSpace(0._RK, 10._RK, 4, .false., .false.) ) ! = [0.00000000, 3.33333325, 6.66666651, 10.0000000] ! (default behavior)
write(*,*) log( genLogSpace(0._RK, 10._RK, 4, .false., .true.) ) ! = [0.00000000, 2.50000000, 5.00000000, 7.50000000]
write(*,*) log( genLogSpace(0._RK, 10._RK, 4, .true., .false.) ) ! = [2.50000000, 5.00000000, 7.50000000, 10.0000000]
write(*,*) log( genLogSpace(0._RK, 10._RK, 4, .true., .true.) ) ! = [1.25000000, 3.75000000, 6.25000000, 8.75000000]
