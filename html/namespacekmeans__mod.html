Perform the Kmeans clustering on the input data set represented by the array Point(nd, np) and return the resulting cluster characteristics as an object of type Kmeans_type.

Parameters
[in]nd: See the description of the runKmeans.
[in]np: See the description of the runKmeans.
[in]nc: See the description of the runKmeans.
[in]Point: See the description of the runKmeans.
[in]InitCenter: See the description of the runKmeans.
[in]niterMax: See the description of the runKmeans.
[in]nszciMax: See the description of the runKmeans.
[in]relTol: See the description of the runKmeans.
Returns
Kmeans : An object of type Kmeans_type containing the properties of the identified clusters. The Errstat component of the output Kmeans will be set to 1 if the number of iterations reaches the input niterMax or its default value before convergence occurs. The Errstat component of the output Kmeans will be set to 2 if more than nzsciMax clustering tries result in clusters with zero members. In both cases in the above, Erroccurred = .true. upon exiting the procedure.
Warning
The input value for niterMax must be larger than 0 since at least one iteration will be performed by the procedure.
This algorithm does not check for the consistency of the input values for nc, nd, and np. It is currently the user's responsibility to ensure that these input values are logical and sensible.

Perform the Kmeans clustering on the input data set represented by the array Point(nd, np).

Parameters
[in]nd: The number of dimensions (features) of the input data array Point.
[in]np: The number of observations in the input data array Point.
[in]nc: The number of clusters to be found via the Kmeans algorithm.
[in]Point: The input array of size (nd,np) of the points to be clustered via the Kmeans algorithm.
[out]Size: A vector of size nc (the number of clusters) containing the sizes of the individual clusters identified.
[out]Center: An array of size (nd,nc) containing the centers of the individual clusters identified.
[out]Membership: A vector of size np each element of which represents the cluster ID to which the point belongs.
[out]NormedPoint: An array of size (nd,np,nc) containing the normalized input array Point(nd,np) w.r.t. each cluster center.
[out]MinDistanceSq: An array of size (np) containing the minimum distances of the points from the cluster centers.
[out]potential: The sum of the distances-squared of all points from their corresponding group centers.
[out]niter: The number of iterations to reach convergence.
[out]nzsci: The number of Zero-Sized Clusters Iterations.
[out]Err: An object of class Err_type containing information about error occurrence.
[in]InitCenter: An input array of size (nd,nc) representing the initial centers of the Kmeans clusters (optional). If not provided, the cluster centers will be initialized via the Kmeans++ algorithm.
[in]niterMax: The maximum number of iterations beyond which it a lack of converge is assumed (optional, default = 1000).
[in]nszciMax: The maximum number of zero-sized cluster iterations beyond which it a lack of converge is assumed (optional, default = 10).
[in]relTol: If the relative difference between two subsequent potentials is below relTol, convergence is assumed (optional, default = 1.e-4).
Warning
The input value for niterMax must be larger than 0 since at least one iteration will be performed by the procedure.
This algorithm does not check for the consistency of the input values for nc, nd, and np. It is currently the user's responsibility to ensure that these input values are logical and sensible.

Find nc initial cluster centers for the Kmeans algorithm, using Kmeans++ recipe.

Parameters
[in]nc: The number of clusters to be found via the Kmeans algorithm (must be > 1, though it is not verified).
[in]nd: The number of dimensions (features) of the input data array Point.
[in]np: The number of observations in the input data array Point.
[in]Point: The input array of size (nd,np) of the points to be clustered via the Kmeans algorithm.
[out]SumPoint: An output array of size (nd,nc) containing the sum of the points in each cluster. When divided by the corresponding output cluster size, it will yield the final updated cluster centers.
[out]Membership: An output array of size (np) representing the IDs of the clusters to which individual points belong.
[out]Size: An output array of size (nc) representing the sizes of the identified clusters.
[out]potential: The sum of the minimum-distances-squared of all points from their corresponding group centers.
Warning
There is no benefit in calling this routine for nc = 1. As such, calling this routine with nc = 1 will lead to undefined behavior.
