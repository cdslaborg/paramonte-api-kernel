90 self%def = 100000_IK
91#endif
92 self%null = NULL_IK
93 self%desc = &
94 "outputChainSize determines the number of non-refined, potentially auto-correlated, but unique, samples &
95 &drawn by the MCMC sampler before stopping " // methodName // ". &
96 &For example, if you specify outputChainSize = 10000, then 10000 unique sample points (with no duplicates) will be drawn &
97 &from the target objective function that the user has provided. &
98 &The input value for outputChainSize must be a positive integer of a minimum value ndim + 1 or larger, &
99 &where ndim is the number of dimensions of the domain of the objective function to be sampled. &
100 &Note that outputChainSize is different from and always smaller than the length of the constructed MCMC chain. &

152 val2char(nd) // ". &
153 &If you don't know an appropriate value for outputChainSize, drop it from the input list. " // &
154 methodName // " will automatically assign an appropriate value to it."//NLC//NLC
