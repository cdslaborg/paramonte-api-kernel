Return the determinant of a given the square matrix of shape (ndim,ndim) via LU factorization.

Parameters
[in]Matrix: The input contiguous array of shape (ndim,ndim) where ndim is size of the input square matrix along each dimension..
[out]failed: The output logical flag that is .true. if the LU decomposition of the Matrix fails. (optional, if missing, the program will halt by executing error stop if an error occurs.)
Returns
determinant : The determinant of the matrix.
Warning
The input Matrix must be square. This condition is only checked when the preprocessor macro CHECK_ENABLED evaluates to true.

Generate an ordered matrix of the input matrix by rearranging the columns corresponding to ColIndx to into the corresponding columns in ColIndxMap, while keeping the rest of the matrix structure intact, such that if it is positive-definite, it remains positive-definite. For example,.

11.0 12.0 13.0 14.0 15.0
0.0 22.0 23.0 24.0 25.0
0.0 0.0 33.0 34.0 35.0
0.0 0.0 0.0 44.0 45.0
0.0 ******* 0.0 0.0 55.0

goes to the following by a ColIndx=[4], ColIndxMap=[5],

11.0 12.0 13.0 15.0 14.0
0.0 22.0 23.0 25.0 24.0
0.0 0.0 33.0 35.0 34.0
0.0 0.0 0.0 55.0 45.0
0.0 0.0 0.0 0.0 44.0
Parameters
[in]rank: The number of columns (or rows) of the input square matrix PosDefMatUpper.
[in]PosDefMatUpper: The input matrix.
[in]lenColIndx: The length of the input ColIndx vector.
[in]ColIndx: An input array of indices indicating the order by which the columns in the input matrix must be rearranged.
[in]ColIndxMap: An input array of length lenColIndxs that indicated the new column index of the corresponding column indices in ColIndx.
Returns
SortedPosDefMatUpper : An ordered matrix of size ( rank, rank ).

◆ getRegresCoef()

PURE subroutine Matrix_mod::getRegresCoef ( integer(IK)  rankPDM,
integer(IK)  rankS11,
integer(IK)  rankS22,
real(RK)  PosDefMat,
real(RK)  RegresCoefMat,
logical, intent(out)  failed,
real(RK)  SchurComplement 
)

Return the the Regression Coefficient Matrix, whose dimension is rankS11 by rankS22, as well as optionally the Schur complement of the S22 block of rank rankS22 of the input matrix PosDefMat of rank rankPDM. For example, if,.

\begin{equation} \texttt{PosDefMat} = \begin{pmatrix} S11 & S12 \\ S21 & S22 \end{pmatrix} \end{equation}

then, the Regression Coefficient Matrix given S22 is: \( S12 \times S22^{-1} \), whose rank is rankS11 by rankS22. The Schur complement of S22 is:

\begin{equation} \texttt{SchurComplement} = S11 - S12 \times S22^{-1} \times S21 ~, \end{equation}

whose rank is rankS11 by rankS11.

Warning
On input, the full matrix PosDefMat must be given.
Remarks
For clarity, note that rankS11 + rankS22 = rankPDM.

pure logical function Matrix_mod::isPosDef ( integer(IK), intent(in)  nd,
real(RK)  Matrix 
)

Return .false. value for isPosDef, if the Cholesky decomposition of the input matrix fails (i.e. matrix is not positive definite), otherwise return .true..

Parameters
[in]nd: The size of the square matrix - nd by nd.
[in,out]Matrix: The input matrix. Note that only the upper triangle of the matrix is used.
Returns
isPosDef : A logical value indicating whether the input matrix is positive-definite.
Warning
Do not call this routine when nd = 1. In such a case, if Matrix(1,1) <= 0, then the matrix is not positive-definite.
