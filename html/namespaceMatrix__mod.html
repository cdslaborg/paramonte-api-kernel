Returns
InvPosDefMat : The full inverse matrix.

Do not call this routine when nd = 1. For nd = 1: InvPosDefMat = 1._RK / InvPosDefMat.

On input, only the upper half of PosDefMat needs to be given.
This routine has the same functionality as the subroutine getInvPosDefMatSqrtDet(), but returns the result as a function output. According to the timing tests, compare_InvMatRoutines_1(), the function version appears to be 15-30% faster than the subroutine version getInvPosDefMatSqrtDet().

◆ getChoLow()

pure subroutine Matrix_mod::getChoLow ( integer(IK), intent(in)  nd,
real(RK)  ChoLowCovUpp,
real(RK)  ChoDia 
)

Return the Cholesky factorization of the input positive-definite matrix.

Parameters
[in]nd: The size of the input square matrix - nd by nd.
[in,out]ChoLowCovUpp: The input square positive-definite (covariance) matrix. Only the upper half and diagonal is needed. The lower half will be rewritten with the lower triangle of the Cholesky Factorization.
[out]ChoDia: The diagonal elements of the Cholesky factorization.
Remarks
If nd = 1, ChoLowCovUpp will not be touched, only sqrt(ChoLowCovUpp) will be written to the output ChoDia.
Warning
If Cholesky factorization fails, ChoDia(1) will be set to -1 to indicate error on return.

Returns in the lower triangle of ChoLowCovUpp, the Cholesky factorization L of \(\texttt{ChoLowCovUpp} = L.L^T\). On input, the upper triangle of ChoLowCovUpp should be given, which remains intact on output.
