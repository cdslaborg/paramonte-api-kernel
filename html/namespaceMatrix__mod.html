pure subroutine Matrix_mod::getChoLow ( integer(IK), intent(in)  nd,
real(RK)  ChoLowCovUpp,
real(RK)  ChoDia 
)

Return the Cholesky factorization of the input positive-definite matrix.

Parameters
[in]nd: The size of the input square matrix - nd by nd.
[in,out]ChoLowCovUpp: The input square positive-definite (covariance) matrix. Only the upper half and diagonal is needed. The lower half will be rewritten with the lower triangle of the Cholesky Factorization.
[out]ChoDia: The diagonal elements of the Cholesky factorization.
Remarks
If nd = 1, ChoLowCovUpp will not be touched, only sqrt(ChoLowCovUpp) will be written to the output ChoDia.
Warning
If Cholesky factorization fails, ChoDia(1) will be set to -1 to indicate error on return.

Returns in the lower triangle of ChoLowCovUpp, the Cholesky factorization L of \(\texttt{ChoLowCovUpp} = L.L^T\). On input, the upper triangle of ChoLowCovUpp should be given, which remains intact on output.

◆ getRegresCoef()

pure subroutine Matrix_mod::getRegresCoef ( integer(IK)  rankPDM,
integer(IK)  rankS11,
integer(IK)  rankS22,
real(RK)  PosDefMat,
real(RK)  RegresCoefMat,
real(RK)  SchurComplement 
)

Return the the Regression Coefficient Matrix, whose dimension is rankS11 by rankS22, as well as optionally the Schur complement of the S22 block of rank rankS22 of the input matrix PosDefMat of rank rankPDM. For example, if,.

\begin{equation} \texttt{PosDefMat} = \begin{pmatrix} S11 & S12 \\ S21 & S22 \end{pmatrix} \end{equation}

then, the Regression Coefficient Matrix given S22 is: \( S12 \times S22^{-1} \), whose rank is rankS11 by rankS22. The Schur complement of S22 is:

\begin{equation} \texttt{SchurComplement} = S11 - S12 \times S22^{-1} \times S21 ~, \end{equation}

whose rank is rankS11 by rankS11.

Warning
On input, the full matrix PosDefMat must be given.
Remarks
For clarity, note that rankS11 + rankS22 = rankPDM.
