type(Wrapper_type) function Bench_mod::Wrapper_type::constructWrapper ( character(*), intent(in)  name,
procedure(exec_proc)  exec,
procedure(exec_proc)  getOverhead 
)

Construct and return an object of type Wrapper_type. This is the constructor of the type Wrapper_type for creating objects to perform benchmarking of the user-specified procedures within the user-provided wrapper function.

Parameters
[in]name: The input name of the procedure that wrapped within the input wrapper procedure pointer.
[in]exec: The input procedure pointer with the abstract interface exec_proc pointing to the user-defined wrapper procedure that calls the arbitrary procedure to be timed.
[in]getOverhead: The input procedure pointer with the abstract interface exec_proc pointing to the user-defined wrapper procedure that calls everything executed within wrapper procedure, except the call to the procedure that is being timed. This procedure pointer will be used to time the overhead due to calling or executing any non-relevant statements within the wrapper (optional. If missing, then a default empty wrapper procedure will be used to compute the overhead).
