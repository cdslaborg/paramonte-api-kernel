Detailed Description

Generate and return the diagonal elements of the input matrix of arbitrary shape.

Parameters
[in]Matrix: The input contiguous array of rank 2 of either
  type real of kind RK32, RK64, or RK128 or
  type complex of kind CK32, CK64, or CK128
  representing the matrix whose diagonal elements is to be retrieved.
[in]ndiag: The input positive integer of default kind IK representing the number of diagonal elements to be extracted from the input Matrix. (optional, default = minval(shape(Matrix) - [srow, scol]) + 1).
[in]srow: The input positive integer of default kind IK representing the starting row of the input Matrix from which the diagonal elements will be extracted (optional, default = 1).
[in]scol: The input positive integer of default kind IK representing the starting column of the input Matrix from which the diagonal elements will be extracted (optional, default = 1).
Returns
Diag : The allocatable output diagonal elements of the input Matrix. It is of the same type and kind as the input Matrix.

Warning
If ndiag is simultaneously present as input argument with srow and scol input arguments, then the conditions size(Matrix, dim = 1) >= srow + ndiag - 1 and size(Matrix, dim = 2) >= scol + ndiag - 1 must hold. These conditions are checked only when the preprocessor flag CHECK_ENABLED=1 is set at build time.
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
