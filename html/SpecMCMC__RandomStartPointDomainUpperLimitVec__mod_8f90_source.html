82  self%desc = &
83  "randomStartPointDomainUpperLimitVec represents the upper boundaries of the cubical domain from which the starting point(s) of &
84  &the MCMC chain(s) will be initialized randomly (only if requested via the input variable randomStartPointRequested. &
85  &This happens only when some or all of the elements of the input variable StartPoint are missing. &
86  &In such cases, every missing value of input StartPoint will be set to the center point between randomStartPointDomainUpperLimitVec &
87  &and randomStartPointDomainLowerLimitVec in the corresponding dimension. &
88  &If RandomStartPointRequested=TRUE (or True, true, t, all case-insensitive), then the missing &
89  &elements of StartPoint will be initialized to values drawn randomly from within the corresponding ranges specified by &
90  &the input variable randomStartPointDomainUpperLimitVec. &
91  &As an input variable, randomStartPointDomainUpperLimitVec is an ndim-dimensional vector of 64-bit real numbers, &
92  &where ndim is the number of variables of the objective function. It is also possible to assign only select values of &
93  &randomStartPointDomainUpperLimitVec and leave the rest of the components to be assigned the default value. &
94  &This is POSSIBLE ONLY when randomStartPointDomainUpperLimitVec is defined inside the input file to "//methodname//". &
95  &For example, having the following inside the input file, \n\n&
96  & randomStartPointDomainUpperLimitVec(3:5) = -100\n\n&
97  & will only set the upper limits of the third, fourth, and the fifth dimensions to -100, or,\n\n&
98  & randomStartPointDomainUpperLimitVec(1) = -100, randomStartPointDomainUpperLimitVec(2) = -1.e6 \n\n&
99  & will set the upper limit on the first dimension to -100, and 1.e6 on the second dimension, or,\n\n&
100  & randomStartPointDomainUpperLimitVec = 3*-2.5e100\n\n&
101  & will only set the upper limits on the first, second, and the third dimensions to -2.5*10^100, while the rest of &
102  &the upper limits for the missing dimensions will be automatically set to the default value.\n\n&
103  &The default values for all elements of randomStartPointDomainUpperLimitVec are taken from the corresponding values in the input &
104  &variable domainUpperLimitVec."
105  end function construct
