real(RK)  logVolUnitBall 
)

Return the probability density function of the Multivariate Uniform distribution whose representative matrix has determinant of logDetInvRepMat.

Parameters
[in]logDetInvRepMat: The natural logarithm of the determinant of the inverse of the representative matrix of the ellipsoidal MVU. Note that logDetInvRepMat = -sum(log(ChoDia)) where ChoDia is the vector of the diagonal elements of the Cholesky factorization of the representative matrix of the ellipsoidal domain of the distribution.
[in]logVolUnitBall: The natural logarithm of the volume of a hyper-sphere of unit-radius in nd dimensions. This number can be computed by calling genLogVolUnitBall(nd).
Returns
logProbDen : The natural logarithm of the PDF of the distribution.

Return the probability density function of the Multivariate Uniform distribution whose representative matrix is given by the input RepMat.

Parameters
[in,out]ChoLowRepMatUpp: The upper triangle and diagonals of the representative matrix of the ellipsoidal support of the distribution. It must be a contiguous matrix of shape (nd,nd) where nd is the number of dimensions of the domain. On output, the lower triangle will be overwritten with the lower-triangle of the Cholesky factorization of the input upper-triangle of the representative matrix, if and only if ChoDia input argument is present.
[out]ChoDia: The diagonal elements of the Cholesky factorization of the input representative matrix of the distribution. (optional, if missing, the lower-triangle of the input ChoLowRepMatUpp will remain intact.)
[out]failed: A logical output which is .true. if the Cholesky factorization fails. (optional. If missing while an error occurs, the program will halt with error stop.)
[in]logVolUnitBall: The natural logarithm of the volume of a hyper-sphere of unit-radius in nd dimensions. This number can be computed by calling genLogVolUnitBall(nd). (**optional*, default = genLogVolUnitBall(nd)).
Returns
logProbDen : The natural logarithm of the PDF of the distribution.
