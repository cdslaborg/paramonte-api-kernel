Detailed Description

This type creates objects for representing MultiVariate Uniform distribution with ellipsoidal support (domain of definition).

Remarks
Unlike other multivariate distributions, the covariance matrix along with its inverse and Cholesky factorization are not components of this type since they already exist as components of the Support component of this type, which is an object of type DomainEllipsoid_type.

Construct an object of type DistMultiUnifEllipsoid_type containing the properties of the specified MultiVariate Uniform (MVU) distribution in ndim dimensions.

An ellipsoidal MVU distribution is specified by the mean of the distribution (or equivalently, the center of its support) as well as its representative matrix. The representative matrix is the matrix \(A\) such that for any point \(X\) within the support of the distribution, the following relationship holds,

\begin{eqnarray*} X^T ~ A^{-1} ~ X \leq 1 ~, \end{eqnarray*}

In such a case, the covariance of the ellipsoidal distribution is given by

\begin{eqnarray*} \text{CovMat} = \frac{A}{p + 2} ~, \end{eqnarray*}

Parameters
[in]ndim: The number of dimensions of the domain.
[in]Mean: The mean (center) of the distribution (optional, default = [(0., i = 1, ndim)]).
[in]RepUpp: The upper triangle and diagonal elements of the representative matrix of the ellipsoidal domain. The lower triangle is not used (optional, default = (ndim + 2) *identity matrix).
[in]isCovUpp: An input logical value. If isCovUpp = .true., then the input RepUpp is assumed to be CovUpp representing the upper triangle of the covariance matrix of the distribution (optional, default = .false.). The input argument isCovUpp is relevant only when RepUpp is also present among the input arguments.
[in]Err: An object of type Err_mod::Err_type containing error information. (optional, if not provided and an error occurs, it will lead to a complete program halt).
Returns
self : An object of type DistMultiUnifEllipsoid_type.
Remarks
If not readily available, an input upper-triangle covariance matrix can be constructed by calling genCovUppFromCorUpp(). Furthermore, the covariance matrix can be converted to a representative matrix by multiplying each element of the covariance matrix with (ndim + 2) where ndim is the number of dimensions of the domain of the distribution.
This routine computes the inverse of the input covariance matrix. As such, if an error occurs, the first element of selfChoLow (selfChoLow(1,1)) will be set to a negative number. Additionally, if the input argument Err is present, an error message will be returned in Errmsg and Erroccurred = .true..
By default, when the input argument RepUpp is missing, the returned ellipsoidal MVU distribution will have an identity representative matrix, meaning that the covariance matrix of the distribution is not identity. To generate a distribution object with identity covariance matrix, pass an identity matrix for RepUpp input argument and set the input argument isCovUpp = .true..
