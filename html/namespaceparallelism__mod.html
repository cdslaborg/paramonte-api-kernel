This is the constructor of the ForkJoin_type class. Return the predicted speedup of the parallel simulation given the input characteristics and timing information of the simulation.

Parameters
[in]processCount: The number of processes in the simulation.
[in]lenProcessID: The length of ProcessID vector.
[in]ProcessID: The vector of process IDs.
[in]successProb: The success probability (the effective acceptance rate per objective function call).
[in]seqSecTime: The timing of the sequential sections of the code.
[in]parSecTime: The timing of the parallel sections of the code.
[in]comSecTime: The timing of the communication sections of the code.
Returns
ForkJoin : An object of class ForkJoin_type containing the parallelization speedup.

Predict the parallel simulation speedup for a range of possible processor counts.

Parameters
[in]successProb: The success probability (the effective acceptance rate per objective function call).
[in]seqSecTime: The timing of the sequential sections of the code.
[in]parSecTime: The timing of the parallel sections of the code.
[in]comSecTimePerProc: The timing of the communication sections of the code per processor.
[in]minMaxNumProc: The minimum number of processes for which the speedup will be computed. It must be at least 1.
[out]Speedup: The vector of speedup values for different counts of processes.
[out]lenSpeedup: The length of the vector Speedup.
[out]maxSpeedupNumProc: The number of processes at which the maximum speedup happens.
[out]maxSpeedup: The maximum speedup for any number of processors.
[out]Err: An object of Err_type class indicating whether and error has occurred upon return.
