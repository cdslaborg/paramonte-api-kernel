real(rk) function statistics_mod::getbetacdf::getbetacdf_dpr ( real(rk), intent(in)  alpha,
real(rk), intent(in)  beta,
real(rk), intent(in)  x 
)

Return the Beta distribution Cumulative Probability Density function (CDF) value.

Parameters
[in]alpha: The first shape parameter of the Beta distribution.
[in]beta: The second shape parameter of the Beta distribution.
[in]x: The point at which the CDF will be computed.
Returns
betaCDF : The Beta distribution CDF value at the given input point.
Warning
If x is not in the range [0,1], a negative value for betaCDF will be returned to indicate the occurrence of error.
The onus is on the user to ensure that the input (alpha, beta) shape parameters are positive.
Remarks
This procedure performs all calculations in DPR (real64) real kind. If performance matters more than 64-bit accuracy, then use the getBetaCDF_SPR for a faster, but less-accurate single-precision results.
Todo:
The efficiency of this code can be improved by making x a vector on input.
