pure integer(IK) function Histogram_mod::bin_RK ( real(RK)  AscendingVector,
real(RK), intent(in)  value 
)

Return the index of the element of the ascending-sorted input vector AscendingVector that is the largest element smaller than or equal to value. The search method is linear.

Parameters
[in]AscendingVector: The input contiguous vector of values.
[in]value: The target value to search for in the input vector.
Returns
binIndex : The lower index of the element of AscendingVector that is the largest element smaller than or equal to value.
Warning
If the input value is smaller than all elements of AscendingVector, then binIndex = 0.
If the input value is larger than all elements of AscendingVector, then binIndex = size(AscendingVector).

Return the lower and upper indices of the elements of the ascending-sorted input vector AscendingVector that sandwich the input value. The search method is linear.

Parameters
[in]AscendingVector: The input contiguous vector of values.
[in]value: The target value to search for in the input vector.
[in]lowerIndex: The lower index of the element of AscendingVector that is the largest element smaller than or equal to value.
[in]upperIndex: The lower index of the element of AscendingVector that is the smallest element larger than value.
Warning
If the input value is smaller than all elements of AscendingVector, then lowerIndex = 0 and upperIndex = 1.
If the input value is larger than all elements of AscendingVector, then lowerIndex = size(AscendingVector) and upperIndex = lowerIndex + 1.

pure integer(IK) function Histogram_mod::getBin ( real(RK), intent(in)  x,
real(RK)  lowerBound,
integer(IK), intent(in)  nbin,
real(RK), intent(in)  binsize 
)

Given the range of the variable x, xmin:xmin+binsize*nbin, and the number of bins, nbin, with which this range is divided, find which bin the input value x falls among [1:nbin] bins. The output ibin is the number that identifies the bin.

Parameters
[in]x: The input value whose bin ID is to be found.
[in]lowerBound: The lower limit on the value of x.
[in]nbin: The number of bins to be considered starting from lowerBound.
[in]binsize: The size of the bins. It must be exactly (xmax - xmin) / nbin.
Returns
ibin : The ID of the bin to which the input value x belongs.
Warning
If x <= xmin or x xmin + nbin * binsize, then ibin = -1 will be returned to indicate error.
Remarks
If bmin < x <= bmax then x belongs to this bin.
Todo:
The performance and interface of this routine can be significantly improved. It is more sensible to pass a contiguous array of bin edges as input instead of lowerBound and binsize. If the input point is not within any bins, an index of zero should be returned.

Return the 1-D histogram (Density plot) of the input vector X. The number of bins in the X range (nxbin) is determined by the user. The range of X, [xmin, xmax], should be also given by the user. The program returns two arrays of Xbin and Density(x) as output.

Parameters
[in]method: The method by which the hist count should be returned:
  • "pdf" : Return the probability density function histogram.
  • "count" : Return the count histogram.
[in]xmin: The minimum of the histogram binning.
[in]xmax: The maximum of the histogram binning.
[in]nxbin: The number of histogram bins.
[in]np: The length of input vector X.
[in]X: The vector of length nxbin of values to be binned.
[out]Xbin: The vector of length nxbin of values representing the bin left corners.
[out]Density: The vector of length nxbin of values representing the densities in each bin.
[out]errorOccurred: The logical output flag indicating whether error has occurred.

Return the 2-D histogram (Density plot) of a set of data points with (X,Y) coordinates. The number of bins in the X and Y directions ([nxbin, nybin]) are determined by the user. The range of X and Y (xmin,xmax,ymin,ymax) should also be given by the user. The program returns three arrays of Xbin, Ybin, and Density(y,x) as the output.

Parameters
[in]histType: The method by which the normalization of the histogram counts should be done:
  • "count" : Return the count histogram.
  • "pdf" : Return the probability density function (PDF) histogram.
  • "pdf(y|x)" : Return the conditional PDF of y given x histogram.
  • "pdf(x|y)" : Return the conditional PDF of x given y histogram.
[in]xmin: The minimum of the histogram binning along the x-axis.
[in]xmax: The maximum of the histogram binning along the x-axis.
[in]ymin: The minimum of the histogram binning along the y-axis.
[in]ymax: The maximum of the histogram binning along the y-axis.
[in]nxbin: The number of histogram bins along the x-axis.
[in]nybin: The number of histogram bins along the y-axis.
[in]np: The length of input vector X.
[in]X: The vector of length nxbin of values to be binned.
[in]Y: The vector of length nybin of values to be binned.
[out]Xbin: The vector of length nxbin of values representing the bin centers.
[out]Ybin: The vector of length nybin of values representing the bin centers.
[out]Density: The array of shape (nybin,nxbin) of values representing the densities per bin.
[out]errorOccurred: A logical output value indicating whether an error has occurred.
