Return the lower triangle Cholesky Factor of the covariance matrix of a set of points in the lower part of ChoLowCovUpp. The upper part of ChoLowCovUpp, including the diagonal elements of it, will contain the covariance matrix of the sample. The output argument ChoDia, contains the diagonal terms of Cholesky Factor. This routine delivers the same functionality of getChoLowCovUpp(). However, it is considerably faster, by a few factors, for nd >> 10. Furthermore, it requires significantly less runtime memory, by about half. The exact amount of speedup depends heavily on the architecture and memory layout of the runtime system.

Parameters
[in]nd: The number of dimensions of the input sample.
[in]np: The number of points in the sample.
[in]Mean: The mean vector of the sample.
[in]Sample: The array of shape (nd,np) containing the sample.
[out]ChoLowCovUpp: The output matrix of shape (nd,nd) whose lower triangle contains elements of the Cholesky factor. The upper triangle of the matrix contains the covariance matrix of the sample.
[out]ChoDia: The diagonal elements of the Cholesky factor.
Todo:
The efficiency of this code can be further improved.
