Return the inverse matrix InvMatLow of an input lower-triangle matrix of shape (nd,nd). The upper triangle of the input matrix will not be used.

Parameters
[in]nd: The size of the input matrix.
[in,out]MatLow: The lower triangle input matrix.
Returns
InvMatLow : The lower-triangular inverse matrix of the input lower-triangle matrix.
Warning
The output is not a full inverse matrix. In particular, the upper-triangle elements of the output matrix are not set to zero.

real(RK)  MatLow,

Return the inverse matrix InvMatLow of an input lower-triangle matrix of shape (nd,nd) whose diagonal elements are given separately as the input Diagonal of length nd. The upper triangle of the input matrix will not be used.

Parameters
[in]nd: The size of the input matrix.
[in,out]MatLow: The lower triangle input matrix (except the diagonal terms which are provided separately).
[in,out]Diagonal: The diagonal elements of the lower triangle matrix MatLow.
Returns
InvMatLow : The lower-triangle inverse matrix of the input lower-triangle matrix.
Warning
The output is not a full inverse matrix. In particular, the upper-triangle elements of the output matrix are not set to zero.
Remarks
This procedure is particularly useful for computing the inverse of the lower-triangle of Cholesky factorization returned by the procedure getChoLow().
