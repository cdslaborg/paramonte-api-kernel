This is the constructor for objects of type InterPoly_type.

The input SortedX and SortedY are used to construct the properties of a polynomial interpolation of degree degree. Once the interpolation coefficients are computed, user can call the returned object's predict() method to interpolate at arbitrary input query points.

\implementation

Parameters
[in]SortedX: The contiguous ascending-sorted array of x-values (nodes) of the interpolation.
[in]SortedY: The contiguous array of y-values corresponding to SortedX.
[in]degree: The \(\text{degree} > 0\) of the polynomial used for interpolation (optional, default = 1).
[out]Err: An object of type Err_type containing error information. On output, Erroccurred must be checked for any occurrences of runtime errors.
Returns
InterPoly : An object of type InterPoly_type containing the interpolation properties.
