pure real(rk) function math_mod::getgammahalfint ( real(rk), intent(in)  positiveHalfInteger)

Return the Gamma function for a half-integer input as real of kind RK.

Parameters
[in]positiveHalfInteger: The input half integer as a real number
Remarks
The equation for half-integer Gamma-function is given as,

\begin{equation} \Gamma \left( \frac{n}{2} \right) = \sqrt \pi \frac{ (n-2)!! }{ 2^\frac{n-1}{2} } ~, \end{equation}

Returns
gammaHalfInt : The Gamma function for a half integer input.
