MatrixBuild_mod::genMatDia Interface Reference

Detailed Description

Generate a diagonal matrix of shape (nrow,ncol) with its diagonal elements set to the input Diag. All off-diagonal elements are set to zero.

Parameters
[in]Diag: The input scalar or contiguous array of rank 1 of either
  type real of kind RK32, RK64, or RK128 or
  type complex of kind CK32, CK64, or CK128
containing the diagonal elements of the output matrix.
[in]nrow: The input positive integer of default kind IK representing the number of rows of the output matrix.
[in]ncol: The input positive integer of default kind IK representing the number of columns of the output matrix.
[in]shift: The input positive integer of default kind IK representing the number of columns the diagonal must be shifted in the the output matrix (optional, default = 0)
Returns
DiagMat : The output diagonal matrix of the same type and kind as the input Diag and shape (nrow,ncol) whose (shifted) diagonal elements are set to the input values. All other matrix elements will be set to zero.
Warning
The size of input Diag must be consistent with the input values for nrow, ncol, and shift (if it is present). This condition is checked for only when the preprocessor flag CHECK_ENABLED=1 is activated at build time.
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
