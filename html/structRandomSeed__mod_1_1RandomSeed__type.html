type(RandomSeed_type) function RandomSeed_mod::RandomSeed_type::constructRandomSeed ( integer(IK)  imageID,
integer(IK)  inputSeed,
logical  isRepeatable,
logical  isImageDistinct 
)

This is the constructor of the RandomSeed_type class. Initialize the seed of the random number generator and return an object of class RandomSeed_type containing the information and methods for setting and resetting the random seed.

Parameters
[in]imageID: The ID of the current process.
[in]inputSeed: The optional scalar integer based upon which the seed of the random number generator will be set (optional).
[in]isRepeatable: The logical flag indicating whether the random number sequence must be repeatable upon each restart (optional).
[in]isImageDistinct: The logical flag indicating whether the random seed must be distinct on each processor from others (optional).
Returns
RandomSeed : An object of class RandomSeed_type containing the information and methods for setting and resetting the random seed.
