real(RK) function, dimension(nd) DomainEllipsoid_mod::genRand::genRandMVU ( integer(IK), intent(in)  nd,
real(RK)  Mean,
real(RK)  ChoLow,
real(RK)  ChoDia 
)

Given the Cholesky Lower triangle and diagonals of a given representative matrix, this function return one point uniformly randomly drawn from inside of an nd-ellipsoid, whose nd elements RandMVU(i), i=1:nd are guaranteed to be in the range:

Mean(i) - sqrt(RepMat(i,i)) < RandMVU(i) < Mean(i) + sqrt(RepMat(i,i))
Parameters
[in]nd: The number of dimensions of the MVU domain.
[in]Mean: The mean vector of the MVU domain.
[in]ChoLow: The Cholesky lower triangle of the representative matrix of the MVU domain.
[in]ChoDia: The diagonal elements of the Cholesky lower triangle of the representative matrix of the MVU domain.
Returns
RandMVU : The randomly generated MVU vector.

Given the Cholesky Lower triangle and diagonals of a given representative matrix, this function return one point uniformly randomly drawn from inside of an nd-ellipsoid, whose nd elements RandMVU(i), i=1:nd are guaranteed to be in the range:

Mean(i) - sqrt(RepMat(i,i)) < RandMVU(i) < Mean(i) + sqrt(RepMat(i,i))
Parameters
[in]nd: The number of dimensions of the MVU domain.
[in]Mean: The mean vector of the MVU domain.
[in]ChoLow: The Cholesky lower triangle of the representative matrix of the MVU domain.
[in]ChoDia: The diagonal elements of the Cholesky lower triangle of the representative matrix of the MVU domain.
[in]np: The number of random vectors to generate.
Returns
RandMVU : The matrix of rank (nd,np) each column of which will be one randomly generated MVU vector.
