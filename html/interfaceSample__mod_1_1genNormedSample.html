Return the normalized sample of shape (nd,np) (default) or (np,nd) with respect to the input mean vector of length nd, where nd represents the number of dimensions (data attributes) of the input Sample and np represents the number of data points in the Sample.

Parameters
[in]Sample: The array of shape (nd,np) or (np,nd) containing the sample.
[in]Mean: The contiguous vector of mean the input sample. If needed, the input Mean can be computed by calling the procedures collectively represented by the generic interface Sample_mod::getMean().
[in]dim: An integer indicating which dimension of the input Sample iterates over the individual data. If dim = 1 or dim /= 2, the input Sample is assumed to have the shape (np,nd). If dim = 2, the input Sample is assumed to have the shape (nd,np). (optional, default = 2.)
[in]tenabled: A logical value that if .true., the output NormedSample will have the transposed shape of the input Sample(:,:). Note that tenabled stands for transposeEnabled (optional, default = .false.).
Returns
NormedSample : The output allocatable normalized sample of shape (np,nd) or (np,nd), depending on the specified values for the input argument tenabled.
Warning
Note the difference in the shape of the input Sample vs. the output NormedSample as determined by dim and tenabled.
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
