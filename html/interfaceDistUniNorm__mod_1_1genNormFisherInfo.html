Detailed Description

Generate the Fisher Information of the Normal distribution.

The Fisher information for the Normal distribution is defined by the following equation,

\begin{equation} \large \mathcal{I}(\mu,\sigma) = \begin{pmatrix} \frac{1}{\sigma^2} & 0 \\ 0 & \frac{2}{\sigma^2} \\ \end{pmatrix} \end{equation}

Parameters
[in]varInv: The input scalar of type real of kind RK32, RK64, or RK128 representing the inverse of the variance of the Normal distribution. The positivity of the input value is checked only if the library is built with the preprocessor macro CHECK_ENABLED=1.
Returns
FisherInfo : The output scalar of the same type and kind as the input varInv representing the Fisher Information matrix of the Normal distribution.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
