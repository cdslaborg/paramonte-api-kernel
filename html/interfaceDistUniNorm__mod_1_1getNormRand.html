Return a Gaussian random scalar or array, optionally with the specified input mean and standard deviation of std. More...

Detailed Description

Return a Gaussian random scalar or array, optionally with the specified input mean and standard deviation of std.

Parameters
[out]Rand: The output random Gaussian scalar, or contiguous array of shape (:) or (:,:) or (:,:,:) of type real of kind RK32, RK64, or RK128.
[in]mean: The input scalar of the same type and kind as Rand, representing the mean of the Normal distribution (optional, default = 0.).
[in]std: The input positive scalar of the same type and kind as Rand, representing the standard deviation of the Normal distribution (optional. If missing, default = 1.. If present, then the input argument mean must be also present).
Warning
The positivity of the input std is checked only when the library is built with the preprocessor macro CHECK_ENABLED. In such a case, the program will halt if the input std argument is negative.

getRandMultiNorm
