Detailed Description

This is the class for generating instances of grid-like Multivariate Normal Distribution mixtures.

Remarks
This class is particularly useful for benchmarking different sampling routines of the ParaMonte library.

Set the DistMultiNormGrid properties and return an object of type DistMultiNormGrid_type containing the objective function with the requested properties.

Parameters
[in]ndim: The number of dimensions of each of the MVN distribution components.
[in]nmode: The number of modes of the MVN mixture grid.
[in]Mean: An array of length (ndim,nmode) representing the centers of the MVN (i.e., the grid nodes). (optional, default = [(((i-1)*elongation, i = 1, ndim), j = 1, nmode)] .
[in]elongation: The single standard deviation of each MVN that is non-unity. Larger values yield more elongated grids (optional, default = 10).
[in]domainMargin: The margin of the domain of the definition of grid in units of elongation (optional, default = 3.5).
Returns
self : An object of type DistMultiNormGrid_type containing the objective function with the requested properties.
