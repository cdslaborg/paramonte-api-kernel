◆ test_construct_1()

logical function Test_DistMultiUnifEllipsoid_mod::test_construct_1

Test the construction of an ellipsoidal MVU distribution with minimal input argument (ndim).

  • The Support component must be correctly set to the output returned by DomainEllipsoid_type(ndim).
  • The upper-triangle and diagonals of ChoLowCovUpp must be set to the covariance matrix of the distribution.
  • The lower-triangle of ChoLowCovUpp and the ChoDia component of the object must represent the Cholesky factorization of the covariance matrix of the distribution.
  • The logDetInvCovMat component of the object must be correctly computed.
  • The Std component of the distribution object must represent the vector of standard deviation of the distribution.
  • The InvCovMat component of the object must represent the inverse covariance matrix of the distribution.
  • The log(PDF) of the distribution must be the inverse of the volume of the distribution support.
  • The logpdf() method of the distribution object must properly return the log(PDF) value.
