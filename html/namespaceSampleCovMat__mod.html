Data Types

interface  genCovUpp
 Generate and return the upper triangle of the covariance matrix of the input shifted, optionally-weighted, sample ShiftedSample whose shape is by default assumed to be (nd,np) or (nd,np) such that nd < np always holds, where nd represents the number of attributes (dimensions) of the sample and np represents the number of data points in the sample. Note that by definition, np > nd must hold. The input sample must be normalized in the sense that its mean is zero. More...

Convert the lower-triangle correlation matrix to the lower-triangle covariance matrix.

Parameters
[in]nd: The rank of the correlation matrix.
[in]StdVec: The input standard deviation vector of length nd.
[in]CorLow: The lower-triangle correlation matrix. The upper-triangle will not be used.
Returns
CovLow : The lower triangle of the covariance matrix. The upper-triangle will NOT be set.
