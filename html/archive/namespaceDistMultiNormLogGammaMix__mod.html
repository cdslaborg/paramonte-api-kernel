Detailed Description

This module contains the classes and procedures for setting up and computing a multivariate mixture of Gaussian and LogGamma distributions with four modes. This is a function that is challenging for samplers and can be used for benchmarking purposes. See: Beaujean, 2013, Initializing adaptive importance sampling with Markov chains.
