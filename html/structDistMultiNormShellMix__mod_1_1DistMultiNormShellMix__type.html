Set the DistMultiNormShellMix properties and return an object of type DistMultiNormShellMix_type containing the objective function with the requested properties.

Parameters
[in]ndim: The number of dimensions of the domain of the distribution.
[in]Mean: A contiguous array of shape (ndim,nshell) representing the centers of the MVN shells. (optional, default = [-3.5, 0., +3.5, 0.], that is, with default nshell = 2.
[in]Radius: A contiguous array of shape (nshell) representing the radii of the MVN shells. Note that if Mean missing, then the user input Radius vector should have a length of 2, since the default mean has a length of (optional, default = 2.).
[in]Width: A contiguous array of shape (nshell) representing the widths (thicknesses) of the MVN shells. Note that if Mean missing, then the user input Radius vector should have a length of 2, since the default mean has a length of (optional, default = 2.).
Returns
self : An object of type DistMultiNormShellMix_type containing the objective function with the requested properties.
