◆ testsamplingcounttarget

integer(ik) paramonte_mod::paramonte_type::testsamplingcounttarget = huge(1_IK)

These variables are exclusively used for testing the deterministic restart functionality of ParaDXXX samplers. This block must not be activated under any other circumstances. Under normal testing conditions (other than testing the restart functionality, selftestSamplingCountTarget > selftestSamplingCounter must always hold. To test the restart functionality under any serial or distributed parallelization schemes, set selftestSamplingCountTarget < chainSize. The simulation will automatically, but gracefully, interrupt when this target value is reached or surpassed.

The external user-specified target count at which the code will break with a simulation interruption error message.
