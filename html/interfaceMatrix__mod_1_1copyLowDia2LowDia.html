Matrix_mod::copyLowDia2LowDia Interface Reference

Detailed Description

Copy the lower-triangle and diagonals of the input from square matrix to the lower-triangle and diagonals of the input matrix to.

This subroutine interface is a shortcut to avoid loops for copying the lower triangle and diagonals of a matrix to another existing one. While writing the loop is simple, since it occurs frequently, these subroutines provide a cleaner solution.

Parameters
[in]from: The contiguous square matrix whose lower-triangle will have to be copied into the destination matrix. All elements of from shall be all reals of either RK32, RK64, RK128 kinds.
[in,out]to: The destination matrix of the same type and kind as from to which the lower-triangle and diagonals of from will be copied.
Warning
The input matrices from and to must be square. This condition is checked only when the preprocessor macro CHECK_ENABLED evaluates to true.
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
