◆ winify()

pure character(:) function, allocatable path_mod::winify ( character(len=*), intent(in)  inputPath)

This procedure is a static method of the class Path_type.
Convert the the input path to the modified path according to the rules of the Windows operating system.

Parameters
[in]inputPath: The input path. If provided, it will overwrite Pathoriginal.
Returns
outputPath : The output modified path which conforms to the rules of the Windows OS.
Warning
This code assumes that the input path is a Linux path. Windows paths like .\(paramonte)\paramonte.nml will be horribly treated by this routine as \( also represents a Linux escape character. The result will be .(paramonte)\paramonte.nml.
This routine strictly assumes that there is no dangling \ in the input Linux path, and if there is, then either it is used to escape the special shell characters, or otherwise, the path is a Windows path.
