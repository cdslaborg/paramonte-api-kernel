integer(IK), intent(in), optional  base 
)

Return the smallest length of a vector that is a power of base and at least base**2 times larger than the input length actualLen.

Parameters
[in]actualLen: The input vector length.
[in]base: The integer-valued base of the exponentiation (optional, default = 2).
Returns
paddedLen : The minimum power-of-base length given actualLen.
Remarks
This method is used to compute the cross-correlation. Usage: actualLen = max( actualLen1, actualLen2 )
Warning
The input values for absoluteValue and base must be both larger than 1.
Remarks
For weighted-data cross-correlation computation, try, actualLen = max( sum(Weight1(1:actualLen1)), sum(Weight2(1:actualLen2)) ).
