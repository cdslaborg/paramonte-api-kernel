pure character(:) function, allocatable String_mod::real2str::num2char_RK128_0D ( real(RK128), intent(in)  value,
character(*), intent(in), optional  format,
integer(IK), intent(in), optional  len 
)

Convert the input value to string, with the requested format, if provided.

Parameters
[in]value: The input integer value.
[in]format: The Fortran IO format to be used when writing the integer value to the string (optional, default = "(g0,:,',')").
[in]len: The length of the output string. If len is missing as input, the blanks on both ends of the output string will be trimmed. (optional, default = automatically determined such that all leading and trailing blank characters are removed)
Returns
string : The output string.
Remarks
This procedure is a static method of the class String_type.
Warning
Note that len, if provided, must be large enough to contain the full input value.
