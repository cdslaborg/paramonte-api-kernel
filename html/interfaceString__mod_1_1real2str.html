Convert an input vector of 64-bit real values to string, with the requested format, if provided.

Parameters
[in]Value: The input vector of real values.
[in]format: The Fortran IO format to be used when writing the real value to the string (optional).
[in]len: The minimum length of the output string, adjusted to the left.
Returns
val2char_RK64_D1 : The output vector of strings each representing one real value in the input vector.
Remarks
This procedure is a static method of the class String_type.
