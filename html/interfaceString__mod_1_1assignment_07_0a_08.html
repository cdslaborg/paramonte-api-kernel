String_mod::assignment(=) Interface Reference

Detailed Description

Assign the contents of the input string2 to the input string1.

Parameters
[in]String1: The input scalar or contiguous array of rank 1 of type Str_type whose contents will be the same as the contents of the input String2 argument upon return.
[in]String2: The input scalar or contiguous array of rank 1 of type Str_type whose contents will be copied to the input String1 argument upon return.
Warning
If String1 is a scalar, then String2 shall be a scalar. If String1 is an array, then String2 shall be either a scalar or an array of the same type and size as String1.
This pure procedures documented herein become impure when the ParaMonte library is compiled with preprocessor macro CHECK_ENABLED=1. By default, pure procedures are pure in release build and impure in debug and testing builds.
