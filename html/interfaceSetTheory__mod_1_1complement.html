Detailed Description

Return an array of elements of SuperSet that are not in Set.

Parameters
[in]Set: The input set whose elements will be subtracted from the SuperSet.
[in]SuperSet: The input SuperSet whose missing elements in Set will be returned.
Returns
Complement : The array of elements of Set that are not in SuperSet. By definition, size(Complement) = size(SuperSet) - size(Set).
Warning
All elements of Set and SuperSet must be unique. The unique elements of any arbitrary vector can be obtained by passing the array to Unique_mod::get().
It is assumed that Set is subset of SuperSet. This condition will NOT be checked in this procedure. If this condition does not hold, the results will not be accurate.
