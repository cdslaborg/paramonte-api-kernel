Detailed Description

Solve the set of nd linear equations Matrix * X = Solution.

Parameters
[in]nd: The size of the square matrix and its LU decomposition - nd by nd.
[in]LU: The LU factorization of the matrix, determined by the routine getPLU.
[in]RowPerm: The permutation vector returned by getPLU.
[in,out]Solution: The right-hand-side vector of length nd. On output, it is completely overwritten by the solution to the system.
Remarks
The input nd, LU, and RowPerm parameters are not modified by this routine and, can be left in place for successive calls with different right-hand sides Solution.
This routine takes into account the possibility that Solution will begin with many zero elements, and is therefore efficient for matrix inversion.

Solve the set of nd linear equations of the form,

\begin{equation} A \cdot X = B \end{equation}

where \(A\) is the coefficient matrix represented by its Pivoted LU decomposition matrix CoefMatPLU, \(X\) is the output solution, and \(B\) is the right-hand-side vector of constants.

Parameters
[in,out]Solution: The input contiguous vector of size nd of type real of kind RK32, RK64, or RK128, representing the right-hand-side \(B\) in the system of linear equations to be solved.   On output, it is completely overwritten by the solution \(X\).
[in]CoefMatPLU: The input contiguous square matrix of shape (nd,nd) of the same type and kind as Solution. This matrix is returned by the call to getPLU.
[out]RowPerm: The output contiguous vector of size nd of type integer of default kind IK containing the pivot indices. This vector is returned by the call to getPLU.
[out]Parity: An output real of the same type and kind as the input CoefMatPLU whose value is +1 if the number of row interchanges is even, otherwise it is -1.
[out]failed: The output logical of default kind that is .true. if a singular matrix is detected, indicating the PLU decomposition failure.
