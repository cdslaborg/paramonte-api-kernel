Set the DistMultiUnif properties and return an object of type DistMultiUnifCube_type containing the properties of the specified MultiVariate Uniform (MVU) distribution in ndim dimensions.

Parameters
[in]ndim: The number of dimensions of the domain of the distribution.
[in]SupportLower: An array of length (ndim) containing the lower limits of the cubic support of the multivariate uniform distribution. (optional, default = [( 0., i = 1, ndim )]).
[in]SupportUpper: An array of length (ndim) containing the upper limits of the cubic support of the multivariate uniform distribution. (optional, default = [( 1., i = 1, ndim )]).
[out]Err: An object of type Err_mod::Err_type containing error flag and information. Possible errors include wrong input optional arguments and the failure of the Cholesky factorization of the covariance matrix. (optional, if missing, the procedure will call error stop upon encountering a fatal error).
Returns
self : An object of type DistMultiUnifCube_type containing the specified distribution and its properties.

Parameters

[in]Point: An array of length ndim at which the objective function will be computed.
Returns
logval : The natural logarithm of the PDF of the distribution.
Remarks
This procedure is only provided for the sake of consistency with the interfaces of other distributions. Otherwise, note that the PDF of uniform distribution is constants for all supported values in the domain.
