Given the integer Array(1:n) return the array Indx(1:n) such that Array(Indx(j)), j=1:n is in ascending order.

Parameters
[in]n: The length of the input vector to be sorted.
[in]Array: The input vector of length n to be sorted.
[out]Indx: The output integer vector of indices of the sorted vector.
[out]Err: An object of class Err_type.
Warning
On return, the value of Err%occurred must be checked for any potential occurrences of errors during sorting.
