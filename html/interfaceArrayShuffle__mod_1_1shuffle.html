ArrayShuffle_mod::shuffle Interface Reference

Perform an unbiased random shuffling of the input array, known as the Knuth or Fisher-Yates shuffle. More...

Detailed Description

Perform an unbiased random shuffling of the input array, known as the Knuth or Fisher-Yates shuffle.

Parameters
[in,out]Array: The input/output contiguous array of shape (:) of either
  type integer of kind IK8, IK16, IK32, or IK64, or
  type complex of kind CK32, CK64, or CK128, or
  type real of kind RK32, RK64, or RK128, or
  type logical of default kind LK, or
  type character of default kind, or
  a scalar assumed-length character of default kind,
  whose elements will be shuffled randomly on return.
