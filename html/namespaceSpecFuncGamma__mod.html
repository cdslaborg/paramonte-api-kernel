Return the upper incomplete Gamma function Q(exponent, lowerLim) evaluated by its continued fraction representation as gammaContFrac.

Parameters
[in]exponent: The exponent in upper incomplete Gamma function Q(exponent, upperLim).
[in]logGammaExponent: This is the log( gamma(exponent) ).
[in]lowerLim: The lower limit of the integration.
[in]tolerance: Represents the relative accuracy (optional).
Returns
gammaContFrac : The incomplete Gamma function Q(exponent, lowerLim) evaluated by its continued fraction representation. Note that gammaContFrac is, by definition, always positive. A -infinity value will be assigned to signal the failure to converge, due to an extremely large exponent or too small ITMAX.
Warning
If the algorithm fails to converge, gammaContFrac will be set to negative infinity on output to signal error.

Return the lower incomplete Gamma function P(exponent, upperLim) evaluated by its series representation as gammaSeries.

Parameters
[in]exponent: The exponent in lower incomplete Gamma function P(exponent, upperLim).
[in]logGammaExponent: The input log( gamma(exponent) ).
[in]upperLim: The upper limit in lower incomplete Gamma function P(exponent, upperLim).
[in]tolerance: The input relative accuracy.
Returns
gammaSeries : The lower incomplete Gamma function P(exponent, upperLim) evaluated by its series representation. Note that the output is always positive unless an error occurs, in which case, it will be set to -infinity. This implicit behavior is to preserve the purity of the procedure. It is the user's responsibility to check for the non-negativity of the output upon return for any failures.
Warning
Do not set tolerance to a number larger than 1, or else the code will crash spectacularly.
If the algorithm fails to converge, gammaSeries will be set to negative infinity on output to signal error. An error occurs only when the series fails to converge.

Return the lower incomplete Gamma function for the specified exponent and upper limit. The input tolerance represents the relative accuracy of the result.

Parameters
[in]exponent: The exponent in lower incomplete Gamma function P(exponent, upperLim).
[in]logGammaExponent: The input log( gamma(exponent) ).
[in]upperLim: The upper limit in lower incomplete Gamma function P(exponent, upperLim).
[in]tolerance: The input relative accuracy (optional, The default is set by getGammaSeries()).
Returns

lowerGamma : The lower incomplete Gamma function for the specified exponent and upper limit. Note that lowerGamma is, by definition, always positive. A negative value is used by the procedure to signal the occurrence of an error. It is the user's responsibility to check the non-negativity of lowerGamma upon return.

Warning
Do NOT set tolerance to a number larger than 1, or else the code will crash spectacularly. Ideally, you should avoid specifying it as an input as the procedure will automatically set it appropriately.
If the algorithm fails to converge, lowerGamma will be set to negative infinity on output to signal error.
Do not set tolerance to a number larger than 1, or else the code will crash spectacularly.
Remarks
logGammaExponent = log_gamma(exponent) where log_gamma() is a Fortran intrinsic function.
Warning
On input, both exponent and upperLim must be positive, otherwise, a lowerGamma = -infinity will be returned to signal error.
