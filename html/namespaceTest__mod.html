subroutine Test_mod::assert ( class(Test_type)  Test,
logical, intent(in)  assertion,
character(*), intent(in), optional  desc 
)

Test the validity of the input assertion and if it does not hold,

  1. in debug and testing modes, call error stop while posting the assertion description desc and,
    1. in release mode, report the test failure and continue with the rest of the tests.
Parameters
[in,out]Test: An object of class Test_type (passed implicitly).
[in]assertion: An input scalar of type logical of default kind LK containing the assertion to be tested.
[in]desc: An input assumed-length character of default kind SK containing a description of assertion to be printed on screen only if assertion = .false. and the tests are run in debug mode.
Remarks
This procedure is a method of the class Test_type.
