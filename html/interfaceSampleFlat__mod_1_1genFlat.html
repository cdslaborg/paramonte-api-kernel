Detailed Description

Generate an equally-weighted (flattened) array of the input weighted Array of rank 1 or 2.

Parameters
[in]Array: The input contiguous array of shape (:) or (:,:) of either type real of kind RK32, RK64, or RK128 or type integer of kind IK8, IK16, IK32, or IK64, containing the weighted array to be flattened.
[in]Weight: The input contiguous array of shape (:) of type integer of default kind IK of the same size as the Array containing the weights of points in the sample. The values of elements of Weight are allowed to be negative or zero, in which case, the corresponding elements will be excluded from the output FlatArray.
[in]sumWeight: The input scalar of type integer of default kind IK representing the sum of all positive non-zero elements of Weight (optional, default = sum(Weight, mask = Weight > 0). If a precomputed value available and present as argument, it can lead to a faster runtime).
Returns
FlatArray : The output array of shape (1:count) of the same type and kind as the input x1 containing the evenly-spaced sequence within the interval specified by x1 and x2.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
Remarks
Setting both fopen = .true. and lopen = .true. will lead to an output LinSpace whose points are centers of the count + 1 equally-sized bins in the interval [x1,x2].
