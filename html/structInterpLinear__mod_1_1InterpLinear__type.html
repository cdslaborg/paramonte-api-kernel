real(RK)  SortedY,
type(Err_type)  Err

Return an objects of type InterpLinear_type for linear interpolation.

Parameters
[in]SortedX: The contiguous ascending-sorted array of x-values (nodes) of the interpolation.
[in]SortedY: The contiguous array of y-values corresponding to SortedX.
[out]Err: An object of type Err_type containing error information. On output, Erroccurred must be checked for any occurrences of runtime errors. (optional. If missing and error occurs, the program will halt with error stop.)
Returns
InterpLinear : An object of type InterpLinear_type containing the interpolation properties.

The input SortedX is the vector of nodes of the linear interpolation, whose elements are sorted in ascending order. The input SortedY is the corresponding function values at the given ascending-sorted interpolation nodes SortedY. are used to construct the properties of a linear interpolation of degree degree. The linear interpolation implemented in this constructor is based on the Lagrange classical formula for linear interpolation. Suppose an input query point \(x\) falls between two nodes \(x_i\) and \(x_{i+1}\) with the corresponding function values \(y_i\) and \(y_{i+1}\) and we wish to estimate the corresponding interpolated value \(y(x)\), which can be computed as,

\begin{equation*} y(x) = \frac {x - x_{i+1}} {x_i - x_{i+1}} y_i + \frac {x - x_{i}} {x_{i+1} - x_{i}} y_{i+1} \end{equation*}

Since there are numNode interpolation nodes, there exist ninterp = numNode - 1 such equations as in the above. To expedite future interpolations based on the input nodes, the two fixed coefficients in the above equation are stored in the Coef component on the object for each equation corresponding to a linear interpolation between two adjacet points.

Once the interpolation coefficients are computed, user can call the predict() method of the returned object to interpolate at arbitrary input query points.


Example usage

program test
use InterpLinear_mod, only: InterpLinear_type
type(InterpLinear_type) :: Interp
Interp = InterpLinear_type(SortedX = [1._RK, 2._RK, 3._RK, 4._RK], SortedY = -[1._RK, 2._RK, 3._RK, 4._RK]**2))
write(*,"(*(g0,:,' '))") "Interpolations = ", Interp%predict(QueryX = [0.5_RK, 1.5_RK, 2.5_RK, 3.5_RK, 4.5_RK])
end
