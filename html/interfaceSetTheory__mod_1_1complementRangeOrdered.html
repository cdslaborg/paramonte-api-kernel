Detailed Description

Return an array whose elements are the values in the range lower:upper that are not in OrderedSet.

Parameters
[in]OrderedSet: The input ordered set whose elements will be subtracted from the superset range lower:upper.
[in]lower: The lower limit of the input range that represents the superset.
[in]lower: The upper limit of the input range that represents the superset.
Returns
Complement : The array of elements of OrderedSet that are not in the input superset range lower:upper. By definition, size(Complement) = (upper - lower + 1) - size(OrderedSet).
Warning
All elements of OrderedSet must be unique and ordered ascending. The unique elements of any arbitrary vector can be obtained by passing the array to Unique_mod::get().
It is assumed that OrderedSet is subset of the range lower:upper. This condition will NOT be checked in this procedure. If this condition does not hold, the results will not be accurate.
