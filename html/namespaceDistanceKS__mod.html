Detailed Description

This module contains the classes and procedures for computing the Kolmogorov-Smirnov statistical distance of a univariate dataset from a given distribution or another univariate dataset.

Return the square of Mahalanobis distances for an row-wise array of complex-valued points.

Parameters
[in]nd: The number of dimensions of the input Point array.
[in]np: The number of points in the input input Point array.
[in]Mean: The mean vector of length nd of the sample.
[in]InvCovMat: The inverse covariance matrix (nd,nd) of the sample.
[in]Point: The (nd,np) array of points whose distances squared from the sample are to computed.
Returns
MahalSq : A vector of length np containing the squares of the Mahalanobis distances of the input points from the sample characterized by the input Mean and InvCovMat.

Return the square of Mahalanobis distance for a single point from a distribution of points with mean Mean and inverse matrix InvCovMat. The output is a scalar variable.

Parameters
[in]nd: The number of dimensions of the input Point.
[in]Mean: The mean vector of the sample.
[in]InvCovMat: The full (symmetric) inverse covariance matrix of the sample.
[in]Point: The Point whose distance from the sample is to computed.
Returns
mahalSq : The Mahalanobis distance squared of the point from the sample characterized by the input Mean and InvCovMat.
