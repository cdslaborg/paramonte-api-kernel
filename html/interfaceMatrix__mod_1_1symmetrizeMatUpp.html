Symmetrize an input upper-triangular square matrix by copying the upper triangle to the lower triangle and copying its diagonals from the input diagonal.

Parameters
[in]nd: The size of the input matrix.
[in,out]UpperSquareMatrix: The input upper, and output symmetric full matrix.
[in]Diagonal: The diagonal elements of the output symmetric full matrix.
Remarks
This procedure is particularly useful for symmetrizing the Cholesky lower triangle and diagonal that getChoUpp() generates.
