Detailed Description

Generate and return a new array containing the original array within which the input Selection has been selected at the specified indices Index of the original array.

Parameters
[in]Array: The output contiguous array of shape (:) of either
  type integer of kind IK8, IK16, IK32, or IK64, or
  type complex of kind CK32, CK64, or CK128, or
  type real of kind RK32, RK64, or RK128, or
  type logical of default kind LK, or
  type character of default kind SK, or
  a scalar assumed-length character of default kind SK,
  containing the sequence of values within which the input Selection must be selected at the specified indices.
[in]Selection: The input scalar or contiguous array of shape (:) of the same type and kind as the input Array containing the selection that must be selected at the specified indices of the input Array.
[in]Index: The input contiguous array of shape (:) of type integer of default kind IK, containing the indices of the input Array where Selection must be selected to construct ArrayNew.
  All elements of Index must have values between 1 and the length of Array plus 1.
  Any selection of Index that is negatively valued will be counted from end of the input Array.
  For example, Index = [2,-1] requests selecting Selection at Array(2) and Array(lenArray).
  To append Insersion to Array, specify an index value of lenArray + 1, for example, Index = [lenArray + 1].
  If any value appears repeatedly and sequentially within Index, then the corresponding number of instances of Selection will be selected sequentially at the specified position within Array.
[in]positive: The input logical of default kind LK indicating whether the elements of Index are all positive (indicating counts from the beginning of Array). Setting positive = .true. will lead to a slightly better runtime performance of the algorithm since a conversion of potentially-negative index values to the corresponding positive values (counting from the beginning of Array will be avoided (optional, default = .false.).
[in]sorted: The input logical of default kind LK indicating whether the selections of the specified input Index are all ascending-ordered. This includes the negative values of Index after they are converted to the corresponding positive indices from the beginning of the input Array. Setting sorted = .true. will lead to a better runtime performance of the algorithm since a call to the sort to sort the index values in ascending order will be avoided. However, the onus will be on the user to guarantee the ascending order of the elements of the input Index (optional, default = .false.).
Returns
ArrayNew : The input contiguous array of the same type, kind, and rank as Array containing the original array   within which the input Selection has been selected at the requested indices of the Array.
  The size of ArrayNew will be,
  size(Array) + size(Index) * size(Selection) if Array is a non-scalar-character and Selection is a vector,
  size(Array) + size(Index) * 1 if Array is a non-scalar-character and Selection is a scalar,
  len(Array) + size(Index) * len(Selection) if both Array and Selection are scalar characters.
