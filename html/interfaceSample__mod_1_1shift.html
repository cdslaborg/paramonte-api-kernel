PURE subroutine Sample_mod::shift::shift_2D ( real(RK)  Sample,
real(RK)  Mean,
integer(IK), intent(in), optional  dim 
)

Return the shifted sample of shape (nd,np) (default) or (np,nd) by the input Amount, a contiguous vector of length nd, where nd represents the number of dimensions (data attributes) of the input Sample and np represents the number of data points in the Sample.

Parameters
[in,out]Sample: The array of shape (nd,np) or (np,nd) containing the sample.
[in]Amount: The contiguous vector of the amount by which the sample must be shifted. For example, if the input Sample is to be shifted such that its new mean is the origin, then Amount = -mean(Sample). If needed, the current mean can be computed by calling the procedures collectively represented by the generic interface Sample_mod::genMean().
[in]dim: An integer indicating which dimension of the input Sample iterates over the individual data. If dim = 1 or dim /= 2, the input Sample is assumed to have the shape (np,nd). If dim = 2, the input Sample is assumed to have the shape (nd,np). (optional, default = 2.)
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
Remarks
For details of the naming conventions, see this page.
For details of naming abbreviations, see this page.
This procedure has the same functionality as Sample_mod::genNormedSample_2D(). However, unlike genNormedSample_2D() which is a function and returns the normalized data in a new allocatable object, this subroutine normalizes the input data in place. This is, in general, more efficient and less memory-consuming as the cost of destroying the original data.


Example usage
Suppose the input Sample is to be shifted toward the origin, that is, its new mean is zero (the origin),

use Sample_mod, only: shift, genMean
use Constants_mod, only: IK, RK
ineteger(IK), parameter :: NDIM = 3_IK
ineteger(IK), parameter :: NSIM = 1000_IK
real(RK) :: Sample(NDIM,NSIM)
call random_number(Sample) ! Sample is centered at Mean = [( 0.5_RK, i = 1, NDIM )]
call shift(Sample, -genMean(Sample)) ! Sample is now centered at the origin(zero).
