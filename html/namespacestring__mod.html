◆ split()

type(charvec_type) function, dimension(:), allocatable string_mod::split ( character(len=*), intent(in)  string,
character(len=*), intent(in)  delim,
integer(ik), intent(out), optional  npart 
)

Split the input string string with the input substitute in the input string and return the result.

Parameters
[in]string: The input string.
[in]delim: The delimiter to be used to split the input string.
[out]npart: The number of substrings resulting from splitting the string (optional).
Returns
Parts : An allocatable array of type CharVec_type containing the split parts of the input string.
Remarks
When delim = "", this routine returns a jagged array of strings, each element of which is one character of the input string.
This procedure is a static method of the class String_type.
This procedure should be preferred over the the legacy implementation splitStr() that is kept only for legacy support. This routine is significantly faster than the legacy implementation and its semantic behavior is identical to the Python3's split() string method, except when delim is empty.

)

Split the input string string with the input substitute in the input string and return the result.

Parameters
[in]string: The input string.
[in]delimiter: The delimiter to be used to split the input string.
[out]nPart: The number of substrings resulting from splitting the string (optional).
Returns
Parts : An allocatable array of type CharVec_type containing the split parts of the input string.
Remarks
This procedure is a static method of the class String_type.
Warning
This algorithm is only kept for archival purposes and should not be used in new development, unless the implications and the behavior of this algorithm are fully understood. Use instead split().
The semantic behavior of this algorithm is different from split(). Furthermore, this algorithm is slower than the alternative implementation in split().
