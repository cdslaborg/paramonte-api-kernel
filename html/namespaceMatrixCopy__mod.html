pure real(RK) function, dimension(nd,nd) MatrixCopy_mod::genLowerFromLower ( integer(IK), intent(in)  nd,
real(RK)  Lower,
logical  diagEnabled 
)

Duplicate and return a lower square triangular matrix while ignoring the upper triangle of the input matrix. Convert a lower square triangular matrix to a lower square triangular matrix.

Parameters
[in]nd: The size of the input matrix.
[in,out]Lower: The input lower square triangular matrix.
[in]diagEnabled: An input logical value. If .true., the diagonal elements of the input matrix will also be copied to the output lower-triangle matrix. (optional, default = .false.)
Returns
LowerCopy : The lower copy of the input lower-triangular matrix.
Warning
Note that only the values in the lower-triangle of the input Lower matrix are copied to the output lower-triangle matrix. Furthermore, the upper-triangle of the output LowerCopy matrix is NOT initialized to any specific value, such as 0.
Remarks
This procedure is developed for the purpose of avoiding the unnecessary copy of redundant data in the upper-triangles of matrices. An example use case is when a copy of the lower-triangle of a Cholesky factorization has to be made.
