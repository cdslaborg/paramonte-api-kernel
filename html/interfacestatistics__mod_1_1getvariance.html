Return the variance of the input vector of values of length np.

Parameters
[in]np: The number of points in the sample.
[in]mean: The mean of the vector.
[in]Point: The array of shape np containing the sample.
[in]Weight: The vector of weights of the points in the sample (optional).
[in]sumWeight: The sum of the vector of weights (optional, if Weight is also missing).
Returns
variance : The variance of the input sample.
Warning
If Weight is present, then sumWeight must be also present. Why? if mean is already given, it implies that sumWeight is also computed a priori.
Remarks
One also use the concise Fortran syntax to achieve the same goal as this function:
mean = sum(weight*point) / sum(weight)
variance = sum( (weight*(point-mean))**2 ) / (sum(weight)-1)

But the above concise version will be slightly slower as it involves three loops instead of two.
