logical, intent(out)  failed

Sort (recursively) the input real array of arbitrary size from the smallest value to the largest and return the result inside the input array.

Parameters
[in,out]Array: The input vector to be sorted.
[out]failed: A logical output that is .true. if the procedure fails.
Warning
On return, the contents of the input array is completely overwritten by the output sorted array.
On return, the value of failed must be checked for any potential occurrences of errors during sorting.

[in,out]Rooter: The contiguous array of length lenLeader to be sorted according to the rearrangement of the elements of Leader.
