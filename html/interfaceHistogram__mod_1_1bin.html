real(RK), intent(in)  value

Return the index of the element of the ascending-sorted input vector AscendingVector that is the largest element smaller than or equal to value. The search method is linear.

Parameters
[in]AscendingVector: The input contiguous vector of values.
[in]value: The target value to search for in the input vector.
Returns
binIndex : The lower index of the element of AscendingVector that is the largest element smaller than or equal to value.
Warning
If the input value is smaller than all elements of AscendingVector, then binIndex = 0.
If the input value is larger than all elements of AscendingVector, then binIndex = size(AscendingVector).
