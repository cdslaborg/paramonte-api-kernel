Detailed Description

Generate a sample of shape (np), or (nd,np) or (np,nd) that is shifted by the specified input Amount along the (optionally) specified axis dim. Here, nd stands for the number of dimensions (data attributes) of the input Sample and np represents the number of data points in the Sample. If the input Amount is the negative of the mean of the sample, then the returned sample will have a mean of zero.

Parameters
[in]Sample: The input contiguous array of type real of kind RK32, RK64, or RK128 of shape (np), (nd,np), or (np,nd) containing the sample.
[in]Amount: The input scalar or the contiguous vector of the same type and kind as Sample representing the amount by which the input sample must be shifted.
If the input Sample is a 1D array, then Amount must be a scalar.
If the input Sample is a 2D array, then Amount must be a vector of size nd.
If the sample is to be shifted toward the origin to have a mean of zero, then the input Amount corresponds to the negative of the current mean of the sample that is returned by procedures collectively represented by the generic interface genMean. Note that the size of the input Amount must be consistent with the size of the input Sample.
If the input argument dim = 2 (default) then, size(Amount) == size(Sample,dim=2) must hold.
If the input argument dim = 1 (default) then, size(Amount) == size(Sample,dim=1) must hold.
[in]dim: An integer indicating which dimension of the input Sample iterates over the individual data points.
If dim = 1 or dim /= 2, the input Sample is assumed to have the shape (np,nd).
If dim = 2, the input Sample is assumed to have the shape (nd,np).
(optional, default = 2. This input argument is available only if the input Sample is a 2D array.)
[in]tenabled: A logical value that if .true., the output ShiftedSample will have the transposed shape of the input Sample(:,:). Note that tenabled stands for transposeEnabled (optional, default = .false.. This input argument is available only if the input Sample is a 2D array).
Returns
ShiftedSample : The output allocatable shifted sample of the same type and kind as Sample.
If the input sample is a 1D array, then ShiftedSample has the same shape and size as that of Sample.
If the input sample is a 2D array, then ShiftedSample has the shape of either (np,nd) or (np,nd), depending on the specified values for the input argument tenabled.
