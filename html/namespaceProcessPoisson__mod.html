subroutine ProcessPoisson_mod::getDistSortedExpDiffWeightKS2 ( integer(IK)  SortedIndex,
real(RK)  DistSortedExpDiff,
real(RK)  logVolUnitBall,
real(RK)  ProbKS,
real(RK)  StatKS,
real(RK)  Lambda,
real(RK), intent(out), optional  alpha,
real(RK), intent(out), optional  beta,
type(Err_type)  Err 
)

Return the weights (the KS-test probabilities) of the the input DistSortedExpDiff resulting from the same exponential distribution. Optionally, also return the vector of KS-statistic corresponding to each element of DistSortedExpDiff. The output ProbKS can be used in the weighted likelihood function of the exponential distribution to robustly infer the parameters of the Gamma posterior of the rate parameter of the exponential distribution from which the input DistSortedExpDiff results. , or if logVolUnitBall is present among the input arguments, return the parameters of the posterior distribution of density of the PPP that yielded the input DistSortedExpDiff.

Parameters
[in]SortedIndex: The input contiguous array of indices of DistSortedExpDiff whose weights will be directly computed. The trivial case is to pass a linear range of indices [(i, i = 1, np)]. The longer the length of Index, the more costly the computations will be, especially if Index contains indices of the later elements of DistSortedExpDiff.
[in]DistSortedExpDiff: The input contiguous array of size (np) of exponentiated-distance differences. This array is returned by getDistSortedExpDiff().
[out]ProbKS: The output contiguous array of size (np) of KS two-sample probabilities corresponding to each element in DistSortedExpDiff (optional).
[out]StatKS: The output contiguous array of size (np) of KS two-sample distance metric corresponding to each element in DistSortedExpDiff (optional).
[out]Lambda: The output contiguous array of size (np) of the sample-size-adjusted KS two-sample test statistic corresponding to each element in DistSortedExpDiff (optional).
[out]alpha: The shape parameter of the Gamma posterior probability density function (PDF) of the density (optional).
[out]beta: The scale parameter of the Gamma posterior PDF function of the density (optional).
[out]Err: An object of class Err_mod::Err_type containing error information.
[in]logVolUnitBall: The natural logarithm of the volume of a unit ball in the dimension of interest nd. This quantity is returned by genLogVolUnitBall(). (optional, if provided, then beta will be multiplied with exp(logVolUnitBall) before return. This is equivalent to the beta parameter of the original PPP leading to the input DistSortedExpDiff. If logVolUnitBall is missing among the input arguments, then the return beta will describe the posterior density of the rate parameter of the exponential model that describes the input DistSortedExpDiff.
Warning
On input, the vector SortedIndex must be sorted in ascending order with unique elements.
Remarks
This method relies on minimizing the KS-test statistic. Specifically, input differences vector of exponentiated distances is passed to the KS algorithm iteratively with larger and larger sample sizes to compute the disparity weights of the points. Then, these weights are used to construct the weighted Gamma likelihood function of the points.
