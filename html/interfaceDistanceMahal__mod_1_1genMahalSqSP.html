Return the square of Mahalanobis distance for a single complex point. The output is a scalar variable.

Parameters
[in]nd: The number of dimensions of the input Point.
[in]Mean: The mean vector of the sample.
[in]InvCovMat: The inverse covariance matrix of the sample.
[in]Point: The Point whose distance from the sample is to computed.
Returns
mahalSq : The Mahalanobis distance squared of the point from the sample characterized by the input Mean and InvCovMat.

Return the square of Mahalanobis distance for a single point from a distribution of points with mean Mean and inverse matrix InvCovMat. The output is a scalar variable.

Parameters
[in]nd: The number of dimensions of the input Point.
[in]Mean: The mean vector of the sample.
[in]InvCovMat: The full (symmetric) inverse covariance matrix of the sample.
[in]Point: The Point whose distance from the sample is to computed.
Returns
mahalSq : The Mahalanobis distance squared of the point from the sample characterized by the input Mean and InvCovMat.
