real(RK), dimension(:,:), allocatable Sample_mod::getCovUpp::getCovUpp ( real(RK)  NormedSample)

Return the upper triangle covariance matrix of the input normalized sample NormedSample whose shape is by default assumed to be (nd,np) or (nd,np) such that nd < np always holds, where nd represents the number of attributes (dimensions) of the sample and np represents the number of data points in the sample. Note that by definition, np > nd must hold. The input sample must be normalized in the sense that its mean is zero.

Parameters
[in]NormedSample: The contiguous array of shape (:,:) containing the sample.
Returns
CovUpp : The upper covariance matrix of shape (nd,nd) of the input normalized sample.
Remarks
If needed, the sample normalization before passing it to this procedure can be done by calling the appropriate function under the generic interface Sample_mod::getNormedSample().
If the full symmetric covariance matrix is needed, it can be obtained by passing the output of this procedure to the subroutine Matrix_mod::symmetrizeUpperMatrix().
