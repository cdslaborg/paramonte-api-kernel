type(ChainFileContents_type) function ParaNest_ChainFileContents_mod::ChainFileContents_type::constructChainFileContents ( integer(IK), intent(in)  ndim,

)

This is the constructor of the class ChainFileContents_type.
Return an object of class ChainFileContents_type given the input specifications.

Parameters
[in]ndim: The number of dimensions of the domain of the objective function.
[in]allocSize: The target allocation size of the chain (optional, default = chainSize).
[in]domainAxesNameList: The list of variable names corresponding to each axis of the domain of the objective function (optional).
[in]chainFilePath: The path to the output chain file (optional, if present, the file contents will be read).
[in]chainFileForm: The file format of the chain file ("binary" vs. "ascii") (optional).
[in]lenHeader: The full length of the first line in the input file (the header line) (optional).
[in]delimiter: The delimiter symbol used in the chain file (optional).
[in]chainSize: The full length of the chain in the output file (optional, will be inferred if missing).
Returns
CFC : An object of class ChainFileContents_type containing the chain.
Warning
If chainFilePath is given, then the rest of the optional arguments must be also given.
