Detailed Description

Generate and return an array of elements of OrderedSuperSet that are not in OrderedSet. This is the same as the ordinary complement() algorithm, however, the orderliness of the input sets leads to faster runtime.

Parameters
[in]OrderedSet: The input set whose elements will be subtracted from the OrderedSuperSet.
[in]OrderedSuperSet: The input OrderedSuperSet whose missing elements in OrderedSet will be returned.
Returns
Complement : The array of elements of OrderedSet that are not in OrderedSuperSet. By definition, size(Complement) = size(OrderedSuperSet) - size(OrderedSet).
Warning
All elements of OrderedSet and OrderedSuperSet must be unique and ordered ascending. The unique elements of any arbitrary vector can be obtained by passing the array to ArrayUnique_mod::genUnique().
It is assumed that OrderedSet is subset of OrderedSuperSet. This condition will NOT be checked in this procedure. If this condition does not hold, the results will not be accurate.
