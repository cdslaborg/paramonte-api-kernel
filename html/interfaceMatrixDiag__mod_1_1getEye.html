MatrixDiag_mod::getEye Interface Reference

Return an identity matrix of shape (nrow,ncol). Note that all diagonal elements of the output matrix are set to 1. More...

Detailed Description

Return an identity matrix of shape (nrow,ncol). Note that all diagonal elements of the output matrix are set to 1.

Parameters
[in]Eye: The output identity matrix of shape (nrow,ncol) whose diagonal elements are set to unity.
It must be either of
  type real of kind RK32, RK64, or RK128 or
  type complex of kind CK32, CK64, or CK128.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.
