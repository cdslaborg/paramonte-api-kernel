Construct an object of type DomainCube_type.

Parameters
[in]ndim: The number of dimensions of the domain.
[in]LimitLower: The vector of size ndim of the lower limits of the domain (optional, default = 0._RK).
[in]LimitUpper: The vector of size ndim of the upper limits of the domain (optional, default = 1._RK).
[in]Err: An object of type Err_mod::Err_type containing error information. (optional, if not provided and an error occurs, it will lead to a complete program halt).
Returns
self : An object of type DomainCube_type.
Warning
The consistency of the input values for LimitLower, LimitUpper are not checked for. It is the user's responsibility to ensure LimitLower < LimitUpper at all circumstances.
