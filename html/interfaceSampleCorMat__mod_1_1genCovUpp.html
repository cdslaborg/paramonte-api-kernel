Detailed Description

Generate and return the upper triangle of the correlation matrix of the input shifted, optionally-weighted, sample ShiftedSample whose shape is by default assumed to be (nd,np) or (nd,np) such that nd < np always holds, where nd represents the number of attributes (dimensions) of the sample and np represents the number of data points in the sample. Note that by definition, np > nd must hold. The input sample must be normalized in the sense that its mean is zero.

Parameters
[in]ShiftedSample: The contiguous array of real type of kind RK32, RK64, or RK128 of shape (:,:) containing the sample. The input sample must be shifted such that its mean is the origin (zero). If needed, the sample can be shifted by computing its mean via genMean and passing its negative together with the sample to either genShifted or getShifted to shift the sample toward the origin.
[in]Weight: The contiguous array of shape (:) of type integer of default kind IK or real of kind RK32, RK64, or RK128 containing the sample weights. The length of Weight must match the length of ShiftedSample along its longest axis. For example, if ShiftedSample has shape (nd,np) and np > nd, then Weight must have the shape (np) (optional, default = 1, that is, all points in the sample have equal weights).
Returns
CovUpp : The upper triangle of the correlation matrix of shape (nd,nd) of the input normalized sample. The type and kind of the output array are the same as that of the input ShiftedSample.

Possible calling interfaces ⛓

use SampleCovMat_mod, only: genCovUpp
CovUpp(1:nd,1:nd) = genCovUpp(ShiftedSample(1:nd,1:np), biased)
CovUpp(1:nd,1:nd) = genCovUpp(ShiftedSample(1:np,1:nd), biased)
CovUpp(1:nd,1:nd) = genCovUpp(ShiftedSample(1:np,1:nd), Weight(1:np), biased)
