MathRange_mod::genRange Interface Reference

Generate evenly spaced integer values at intervals of size step from start to stop. More...

Detailed Description

Generate evenly spaced integer values at intervals of size step from start to stop.

Parameters
[in]start: The input scalar of type integer of kind IK8, IK16, IK32, or IK64, representing the start point of the output Range.
[in]start: The input scalar of the same type and kind as start representing the end point of the output Range.
[in]step: The non-zero input scalar of the same type and kind as start representing the size of the interval between adjacent values in the output Range (optional, default = 1).
Returns
Range : The output array of shape (1 : 1 + (stop - start) / step) of the same type and kind as the input argument start containing the evenly-spaced sequence within the interval specified by start and stop. Note that the specific value of stop may not necessarily be the ending value in the sequence.

use MathRange_mod, only: genRange
