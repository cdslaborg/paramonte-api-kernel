Return the quantiles of an input sample of points, given the input quantile probabilities.

Parameters
[in]Point: The contiguous vector of length np representing the input sample.
[in]SortedQuantileProbability: The contiguous sorted ascending-order vector of probabilities at which the quantiles will be returned.
[out]Quantile: The output vector of length nq, representing the quantiles corresponding to the input SortedQuantileProbability probabilities.
[out]failed: The output logical flag indicating if an error occurred during sorting of the input set of points.
[in]Weight: The vector of length np representing the weights of the points in the input sample.
[in]sumWeight: The sum of the vector weights of the points: sum(Weight).
