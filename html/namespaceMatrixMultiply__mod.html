Return the product of two matrices. The product of two matrices is defined by c(i,j) = a(i,1)*b(1,j) + a(i,2)*b(2,j) + ... + a(i,n)*b(n,j).

Remarks
There is not reason to use this routine as the Fortran intrinsic already provides an optimized implementation of matrix multiplication via matmul(). It is present here only for archival and legacy reasons.
