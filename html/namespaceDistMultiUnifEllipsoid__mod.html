Construct an object of type DistMultiUnifEllipsoid_type containing the properties of the specified MultiVariate Uniform (MVU) distribution in ndim dimensions.

Parameters
[in]ndim: The number of dimensions of the domain.
[in]Mean: The mean (center) of the distribution (optional, default = A vector of all zeros).
[in]CovUpp: The upper triangle and diagonal elements of the covariance matrix of the ellipsoidal domain. The lower triangle is not used (optional, default = Identity matrix).
[in]Err: An object of type Err_mod::Err_type containing error information. (optional, if not provided and an error occurs, it will lead to a complete program halt).
Returns
self : An object of type DistMultiUnifEllipsoid_type.
Remarks
If not readily available, the input covariance matrix CovUpp can be constructed by calling getCovUppFromCorUpp().
This routine computes the inverse of the input covariance matrix. As such, if an error occurs, the first element of selfChoLow (selfChoLow(1,1)) will be set to a negative number. Additionally, if the input argument Err is present, an error message will be returned in Errmsg and Erroccurred = .true..

Returns

Remarks
