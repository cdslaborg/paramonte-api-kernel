◆ num2str_IK16_0D()

pure character(:) function, allocatable String_mod::int2str::num2str_IK16_0D ( integer(IK16), intent(in)  value,
character(*), intent(in), optional  format,
integer(IK), intent(in), optional  strlen 
)

Convert the input value to string, with the requested format, if provided.

Parameters
[in]value: The input integer value.
[in]format: The Fortran IO format to be used when writing the integer value to the string (optional, default = "(g0,:,',')").
[in]strlen: The length of the output string. If strlen is missing as input, the blanks on both ends of the output string will be trimmed. (optional, default = automatically determined such that all leading and trailing blank characters are removed)
Returns
string : The output string.
Remarks
This procedure is a static method of the class String_type.
Warning
Note that strlen, if provided, must be large enough to contain the full input value.
