pure real(RK) function Sample_mod::getMean::getMean_1D ( real(RK)  Sample)

Return the mean of the input 1-dimensional Sample.

Parameters
[in]Sample: The array of shape (nd,np) or (np,nd) containing the sample, where nd is the number of attributes of data in the sample and np is the number of dtaat points in the sample.
Returns
mean : The output mean of the sample.
Remarks
Note that this procedure is implemented only for the sake of comprehensiveness and completeness. The mean of any one-dimensional sample can be readily computed via the Fortran intrinsic routine sum(): mean = sum(Sample) / size(Sample) assuming the data points have equal weight.

integer(IK), intent(in), optional  dim 
)

Return the mean of the input Sample.

Parameters
[in]Sample: The array of shape (nd,np) or (np,nd) containing the sample, where nd is the number of attributes of data in the sample and np is the number of dtaat points in the sample.
[in]dim: An integer representing the dimension (1 or 2) along which data is stored in Sample. For example, dim = 1 means that Sample has the shape (np,nd), whereas dim = 2 implies a shape (nd,np) (optional, default = 2).
Returns
Mean : The output mean vector of length nd.
Warning
Note the default assumed shape of the input argument Sample(:,:).
Remarks
Note that this procedure is implemented only for the sake of comprehensiveness and completeness. The mean of any two-dimensional sample can be simply computed via the Fortran intrinsic routine sum(): Mean = sum(Sample, dim = 1) / size(Sample, dim = 1) assuming the first dimension represents the rows of data. Mean = sum(Sample, dim = 2) / size(Sample, dim = 2) assuming the second dimension represents the columns of data.
Warning
This procedure is impure when the library is compiled with macro CHECK_ENABLED=1. By default, it is pure in release build and impure in debug and testing builds.

Return the mean of the input 1-dimensional weighted Sample.

Parameters
[in]Sample: The array of shape (nd,np) or (np,nd) containing the sample, where nd is the number of attributes of data in the sample and np is the number of dtaat points in the sample.
[in]Weight: The vector of same length as Sample containing the weights of points in the sample.
Returns
mean : The output mean of the sample.
Remarks
Note that this procedure is implemented only for the sake of comprehensiveness and completeness. The mean of any one-dimensional sample can be readily computed via the Fortran intrinsic routine sum(): mean = sum(Weight*Sample) / sum(Weight) assuming the data points are weighted.
The WIK in the procedure name stands for Weight of Integer Kind.
