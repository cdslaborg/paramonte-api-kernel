MathMinMax_mod::getMinMax Interface Reference

Return the minimum and maximum of the two input scalar values a and b in a and b, respectively, on output. of the two input numbers in the first and second elements, respectively. More...

Detailed Description

Return the minimum and maximum of the two input scalar values a and b in a and b, respectively, on output. of the two input numbers in the first and second elements, respectively.

Parameters
[in]a: The input scalar of either
  type real of kind RK32, RK64, or RK128 or
  type complex of kind CK32, CK64, or CK128 or
  type integer of kind IK8, IK16, IK32, or IK64 or
  allocatable scalar of type character of default kind.
  On output, its value will be overwritten with min(a,b)
[in]b: The input scalar of the same type and kind as the input a.
  On output, its value will be overwritten with max(a,b)
Remarks
This procedure is pure.
See also
genMinMax
