Sort (recursively) the input real array of arbitrary size from the smallest value to the largest and return the result inside the input array.

Parameters
[in]np: The length of the input vector to be sorted.
[in,out]Point: The input vector to be sorted.
[out]Err: An object of class Err_type.
Warning
On return, the contents of the input array is completely overwritten by the output sorted array.
On return, the value of Erroccurred must be checked for any potential occurrences of errors during sorting.
