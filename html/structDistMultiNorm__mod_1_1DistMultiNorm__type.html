Set the DistMultiNorm properties and return an object of type DistMultiNorm_type containing the properties of the specified MultiVariate Normal (MVN) distribution in ndim dimensions.

Parameters
[in]ndim: The number of dimensions of the MVN distribution.
[in]Mean: An array of length ndim representing the mean of the MVN distribution (optional, default = [(0._RK, i = 1, ndim)].
[in]CovUpp: An array of shape (ndim,ndim) containing the upper triangle (including the diagonal elements) of the covariance matrix of the MVN distribution (optional, the default is the identity matrix).
[in]Std: An array of length ndim containing the standard deviation of the MVN distribution along each dimension. (optional, default = [(1._RK, i = 1, ndim)]. Note that only CovUpp or Std should be provided as input and not both. If both arguments are present, the procedure will return with an error message or fatally stop).
[out]Err: An object of type Err_mod::Err_type containing error flag and information. Possible errors include wrong input optional arguments and the failure of the Cholesky factorization of the covariance matrix. (optional, if missing, the procedure will call error stop upon encountering a fatal error).
Returns
self : An object of type DistMultiNorm_type containing the specified MVN distribution and its properties.
